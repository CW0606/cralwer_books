<h1 id="chapter_title_177988"><strong>（四十四）</strong></h1>
<p class="info">
分类：<a class="cate" href="/novel-list-23.html" target="blank">文学小说</a><span class="divide">|</span>作者：<a class="name" href="/pdnovel.php?mod=search&amp;ac=author&amp;srchtxt=魏巍&amp;searchsubmit=yes"  target="_blank">魏巍</a><span class="divide">|</span>发布时间：2015-04-23 23:34:04<span class="divide">|</span></p>
<div class="page-body">
<div class="page-content">
<p>这些天，来自中央军委的电报，差不多都有“迅速”二字。什么“迅速”前进，“迅速”占领，“限令”到达等等，足可推测出统帅部的急迫心情。他们的意图很明显，就是乘各路敌军到来之前，抢先渡河，以免陷入石达开的不幸境地。但是，按照命令每天要走一百二十里路的艰辛的战士们，仍未能赶到敌军的前面。在红军到达前，刘文辉、杨森等部，已经沿大渡河布防就绪。由富林至泸定桥以及由泸定桥至康定，都由刘文辉的二十四军负责；富林以下至金口，由杨森的二十军防守。五月二十三日，刘湘部装备精良的王泽浚旅也自成都赶到，重点坚守富林，二十四军北移，这样兵力就更厚了。 <br/>　　一心想当骆秉章的杨森，到达汉源不久，即到大渡河沿线视察。这天他到了富林，王泽浚亲自把他迎到旅部，因为按照蒋介石的命令，王泽浚也统归杨森指挥。 <br/>　　王泽浚是四川军阀王缵绪的儿子。王缵绪在刘湘手下当师长，王泽浚就在他父亲兼师长的领导下当旅长，并且兼成都市的城防司令。他出身将门，少年得志，颇有一点不可一世的派头。他这个旅有三个团共六千人，不仅人员充实，且装备精良。配备的迫击炮、轻重机枪、冲锋枪、掷弹筒都比较新式。这次又是蒋介石亲自点名要他星夜驰赴富林，更是声价十倍。在杨森这位老前辈面前，他自然拘于礼法，表现出一副谦恭样子，但内心深处却自命不凡。 <br/>　　“军座，您这次刚到前线，就来敝部视察，真可谓不辞劳苦哇！” <br/>　　“贤侄，你说到哪里去了！”杨森老味十足地说，“这次大渡河会战，委员长亲自给我打电报，要我做当代的骆秉章，我受蒋公如此重托，咋个敢怠慢呢！” <br/>　　自从蒋介石打了这封电报，杨森已经是三句话离不开骆秉章了。王泽浚听了，不自觉地撇了撇嘴，露出了不以为然的笑容： <br/>　　“听说，刘文辉军长到了汉源，也说要当骆秉章呢！” <br/>　　杨森哈哈大笑： <br/>　　“哈哈哈……刘文辉，他也想当骆秉章！哈哈哈……” <br/>　　王泽浚见杨森如此狂妄自许，心中不悦，就笑着说： <br/>　　“这次各路人马，齐集大渡河，恐怕都要显显神通，还说不定鹿死谁手呢！” <br/>　　杨森一听这话，觉得颇有一点不敬之意，他那雷公嘴立刻就凸出来了。但又不好发作，就说： <br/>　　“委员长的三条命令，你们都看到了吗？” <br/>　　“都执行了。”王泽浚说，“船都弄到这边来了；一切可供造船、修桥的材料，甚至竹片、木片，都收走了；还清扫了射界。” <br/>　　“河那边的房子呢？” <br/>　　“也都烧了。” <br/>　　“不，不，”杨森镇着脸说，“贤侄，你这项事情可做得不大彻底，我刚才看到对岸，有许多村庄、房子还没有动，这是要留给共军利用吗？” <br/>　　王泽浚面红耳赤，立刻把一个团长找来，气愤愤地责问道： <br/>　　“在你那个防区里，扫清射界的事情完成了吗？” <br/>　　“完成了一部分。”团长怯生生地回答。 <br/>　　“你说的是个啥子？” <br/>　　“是这样，旅长，老百姓哭得厉害，一跪一大片，士兵们也不愿干。” <br/>　　“哦，老百姓一哭，我的命令你就不执行了？……你这个窝囊废！” <br/>　　“旅长，你别这么说，”团长反抗了，“就是你在那里也不好办。” <br/>　　一句话，把王泽浚激怒了，更何况是在外军军长面前？他立刻从里间屋墙上取下马鞭子来，大声骂道： <br/>　　“你这个不服从命令的东西！我要好好教训教训你。”说着劈头盖脸，连续抽了下来。 <br/>　　这王泽浚是有名的专横跋扈，经常以马鞭抽打部属，就是团长也在所不免。今天他觉得部下伤了自己的面子，自然特别气愤。 <br/>　　杨森见王泽浚这般光景，知道是对自己撒气，就撇撇嘴说： <br/>　　“算了，算了，现在还来得及，叫他去完成也就是了。” <br/>　　王泽浚把马鞭往地上一甩，说： <br/>　　“今天要不是杨军长讲情，我就揍死你！” <br/>　　那个团长忍气吞声，捂着脸上两条赤红色的血痕退出去了。 <br/>　　这时，忽报本地羊土司前来晋谒。 <br/>　　这里说的羊土司，名羊仁安，是大渡河沿岸有名的土著势力，还挂着富林垦殖司令一个官名。他的势力范围是安顺场下游到富林一带。大渡河的另一土著势力，是安顺场的彝务总指挥部营长赖执中，其势力范围是从安顺场起到上游河道七场。这两个封建霸主，在各自的势力范围内为所欲为，生杀予夺，说一不二。大渡河的流水，每年雨季都要冲刷出一种稀罕宝物，名叫香杉。它是埋没在地下的一种杉木，经过千百年水土的浸蚀，渐渐变成一种紫郁郁的异常坚硬的木质，就再也不会腐坏了。梦想不朽的上等人就把它作为做棺材的理想材料，称为“建板”。这种价格极为昂贵的天财地宝，也只有他两人才能享用。不管在何处发现，都要交给他们。在交给他们之前，还要负责看管，如果损坏丢失，就难免倾家荡产，连身家性命都难保了。 <br/>　　自红军向大渡河进军以来，羊仁安早就坐不住了。为了保住自己这个小小王国的安全，他忙得手脚不沾地，慰问来往军队，商讨地方势力如何与军队配合，真是不遗余力。凡是从这里经过的来往军官，他都要宴请一番。王泽浚的到来，他已宴请过一次，今天赫赫有名的杨将军到来，岂是可以疏忽的？所以他穿着轻飘飘的一身绸衫，很快就跑来了。 <br/>　　他一见杨森，就连跑几步，抓住杨森的手说： <br/>　　“杨军长，你是坐飞机来的，还是坐火车来的，真想不到你来得这么快哩！” <br/>　　杨森哈哈一笑，算作回答。 <br/>　　羊仁安坐下来，又望着杨森说： <br/>　　“说实在话，你没来以前，我这心就象十五只吊桶打水，七上八下，您这一来，我这心就定下来了。” <br/>　　杨森冲着王泽浚一笑： <br/>　　“我们的少年将军不是早来了嘛！” <br/>　　“不管小将、老将，还要名将指挥嘛！” <br/>　　杨森心里得到某种满足，哈哈大笑。 <br/>　　羊仁安见是火候，就笑着说： <br/>　　“寒舍备了一点便饭，给军长接风。请军长一定赏光。王旅长一定作陪。” <br/>　　杨森笑着说： <br/>　　“我初来乍到，寸功未立，怎好无功受禄？” <br/>　　王泽浚也笑着说： <br/>　　“我已经叨扰过了。” <br/>　　羊仁安站起来，满脸是笑地说： <br/>　　“你们谁也不要见外，我们马上就走！” <br/>　　杨森、王泽浚、羊仁安骑上快马，后面跟着随从，沿着大渡河边向西驰去。 <br/>　　宴会在羊仁安相当阔绰的宅第举行。宅第的牢固一如小小的城堡，宴会的珍馐美味也使杨森大为惊异。他想不到这小山沟里还有这样的所在。 <br/>　　宴席设在一座小楼上，摆设精致，宽敞明亮，窗外下面就是大渡河的惊涛骇浪。羊仁安端起酒杯，举到杨森胸前，郑重说道： <br/>　　“下面就是长毛贼石达开覆亡之处。这次共匪北窜，已经到了绝境，是再也逃不过了。看来今天的骆秉章就是将军您了。” <br/>　　杨森一听这话，立时甜到心里，笑在脸上，把满满一大杯灌了下去，抹抹嘴说： <br/>　　“那倒要大家多协助了。” <br/>　　王泽浚脸上刚刚露出一点不悦之色，羊仁安已把酒端到胸前，说： <br/>　　“王旅长少年英俊，才气不凡，杨将军这次是骆秉章，你就是亲自捉石达开的唐友耕了！” <br/>　　一句话也说得这位少年将军眉开眼笑，一仰脖儿把一大杯灌了下去。 <br/>　　小楼上气氛热烈，笑语声喧。杨森一连饮了几大杯，忽然停住杯问： <br/>　　“羊土司，听说你们这里出一种啥子香杉很有名气？”“哦，是的，是的，”羊土司笑着说，“本地没啥子好东西，就是这个还算一宝。可是这一带刁民见钱眼开，一遇上这种木头就窝藏起来，亏得我好好惩治了几个，每年才能收到几根。” <br/>　　说到这里，又笑嘻嘻地说： <br/>　　“军座，您是不是需要一点？” <br/>　　“不不，”杨森连忙摇手，“我不过听到家母说过这种材料。” <br/>　　“这个，我回来找人送到司令部去。” <br/>　　这时，不知谁喊了一句： <br/>　　“火！起火了！” <br/>　　大家抬头向窗外一看，大渡河南岸一带村庄，已经冒起一片黑烟，成群的老百姓从村庄里逃向村外，并且传来隐隐的哭叫之声。 <br/>　　杨森点点头说： <br/>　　“好，好，已经开始清扫射界了！” <br/>　　“这些老百姓就是奴隶性！”王泽浚说，“其实早就通知他们了嘛，就硬是不动。” <br/>　　“咳，到处都是一样。” <br/>　　说过，大家又一齐举起杯来。</p></div>
