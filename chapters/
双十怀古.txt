<h1 id="chapter_title_177988"><strong>双十怀古</strong></h1>
<p class="info">
分类：<a class="cate" href="/novel-list-22.html" target="blank">文学作品</a><span class="divide">|</span>作者：<a class="name" href="/pdnovel.php?mod=search&amp;ac=author&amp;srchtxt=鲁迅&amp;searchsubmit=yes"  target="_blank">鲁迅</a><span class="divide">|</span>发布时间：2014-10-06 20:58:10<span class="divide">|</span></p>
<div class="page-body">
<div class="page-content">
<p>　——民国二二年看十九年秋史　癖 <br/>　　小引 <br/>　　要做“双十”（２）的循例的文章，首先必须找材料。找法有二，或从脑子里，或从书本中。我用的是后一法。但是，翻完“描写字典”，里面无之；觅遍“文章作法”，其中也没有。幸而“吉人自有天相”，竟在破纸堆里寻出一卷东西来，是中华民国十九年十月三日到十日的上海各种大报小报的拔萃。去今已经整整的三个年头了，剪贴着做什么用的呢，自己已经记不清；莫非就给我今天做材料的么，一定未必是。但是，“废物利用”——既经检出，就抄些目录在这里罢。不过为节省篇幅计，不再注明广告，记事，电报之分，也略去了报纸的名目，因为那些文字，大抵是各报都有的。 <br/>　　看了什么用呢？倒也说不出。倘若一定要我说，那就说是譬如看自己三年前的照相罢。 <br/>　　十月三日 <br/>　　江湾赛马。 <br/>　　中国红十字会筹募湖南辽西各省急振。 <br/>　　中央军克陈留。 <br/>　　辽宁方面筹组副司令部。 <br/>　　礼县土匪屠城。 <br/>　　六岁女孩受孕。 <br/>　　辛博森伤势沉重。 <br/>　　汪精卫到太原。 <br/>　　卢兴邦接洽投诚。 <br/>　　加派师旅入赣剿共。 <br/>　　裁厘展至明年一月。 <br/>　　墨西哥拒侨胞，五十六名返国。 <br/>　　墨索里尼提倡艺术。 <br/>　　谭延闿轶事。 <br/>　　战士社代社员征婚。 <br/>　　十月四日 <br/>　　齐天大舞台始创杰构积极改进《西游记》，准中秋节开幕。 <br/>　　前进的，民族主义的，唯一的，文艺刊物《前锋月刊》创刊号准双十节出版。 <br/>　　空军将再炸邕。 <br/>　　剿匪声中一趣史。 <br/>　　十月五日 <br/>　　蒋主席电国府请大赦政治犯。 <br/>　　程艳秋登台盛况。 <br/>　　卫乐园之保证金。 <br/>　　十月六日 <br/>　　樊迪文讲演小记。 <br/>　　诸君阅至此，请虔颂南无阿弥陀佛……大家错了，中秋是本月六日。 <br/>　　查封赵戴文财产问题。 <br/>　　鄂省党部祝贺克复许汴。 <br/>　　取缔民间妄用党国旗。 <br/>　　十月七日 <br/>　　响应政府之廉洁运动。 <br/><br/>　　津浦全线将通车。 <br/>　　平津党部行将恢复。 <br/>　　法轮殴毙栈伙交涉。 <br/>　　王士珍举殡记。 <br/>　　冯阎部下全解体。 <br/>　　湖北来凤苗放双穗。 <br/>　　冤魂为厉，未婚夫索命。 <br/>　　鬼击人背。 <br/>　　十月八日 <br/>　　闽省战事仍烈。 <br/>　　八路军封锁柳州交通。 <br/>　　安德思考古队自蒙古返北平。 <br/>　　国货时装展览。 <br/>　　哄动南洋之萧信庵案。 <br/>　　学校当注重国文论。 <br/>　　追记郑州飞机劫。 <br/>　　谭宅挽联择尤录。 <br/>　　汪精卫突然失踪。 <br/>　　十月九日 <br/>　　西北军已解体。 <br/>　　外部发表英退庚款换文。 <br/>　　京卫戍部枪决人犯。 <br/>　　辛博森渐有起色。 <br/>　　国货时装展览。 <br/>　　上海空前未有之跳舞游艺大会。 <br/>　　十月十日 <br/>　　举国欢腾庆祝双十。 <br/>　　叛逆削平，全国欢祝国庆，蒋主席昨凯旋参与盛典。津浦路暂仍分段通车。 <br/>　　首都枪决共犯九名。 <br/>　　林埭被匪洗劫。 <br/>　　老陈圩匪祸惨酷。 <br/>　　海盗骚扰丰利。 <br/>　　程艳秋庆祝国庆。 <br/>　　蒋丽霞不忘双十。 <br/>　　南昌市取缔赤足。 <br/>　　伤兵怒斥孙祖基。 <br/>　　今年之双十节，可欣可贺，尤甚从前。 <br/>　　结语 <br/>　　我也说“今年之双十节，可欣可贺，尤甚从前”罢。十月一日。 <br/>　　附记：这一篇没有能够刊出，大约是被谁抽去了的，盖双十盛典，“伤今”固难，“怀古”也不易了。十月十三日。 <br/>　　　　　　　<br/>===============================================================<br/><br/>　　（１）　本篇收入本书前未能在报刊发表。 <br/>　　（２）　“双十”　即双十节。一九一一年十月十日武昌起义后，建立了中华民国，一九一二年九月二十八日临时参议院决定以十月十日为国庆节。蒋介石于一九二七年窃取政权后，仍以“双十”为国庆节。 </p></div>
