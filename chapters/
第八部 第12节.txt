<h1 id="chapter_title_177988"><strong>第八部 第12节</strong></h1>
<p class="info">
分类：<a class="cate" href="/novel-list-23.html" target="blank">文学小说</a><span class="divide">|</span>作者：<a class="name" href="/pdnovel.php?mod=search&amp;ac=author&amp;srchtxt=周大新&amp;searchsubmit=yes"  target="_blank">周大新</a><span class="divide">|</span>发布时间：2015-04-25 22:32:23<span class="divide">|</span></p>
<div class="page-body">
<div class="page-content">
<p>卓月按照自己的理解写完了安留岗方形土坛的发掘结论之后，迟迟没有将结论送交文化局更没有交《考古新发现》杂志发表。她心里总觉得有点拿不准，总感到其中填充的想象多了点。 <br/>　　那天上午她给学生们讲究历史课后，又不由自主地出了校门向安留岗走去。这是一次没有事先计划的行动，她只是想再去看看那个方形土坛，再看看历史留下的那个难猜的令她苦恼的谜。 <br/>　　用石棉瓦搭起的棚子遮盖着的方形土坛，在这些年间未受风雨的剥蚀，用奇怪颜料划出的网形图案依然清晰可辨。卓月站在坛前默然凝视着这个前朝遗迹，忽然想到，史书上说到祭祀的坛时说：“土基三尺，阶三等曰坛”，这会不会是一座用于祭祀的祭坛？不太可能！她很快又摇头否定了这个想法。——倘是祭坛，坛上怎可摆棺材和死人的尸骨？东汉时可是早就废止了杀人祭祀的习俗！ <br/>　　也许还是自己当初的判断正确，这只是一座坟墓，是王康为爱女王文蕊设计的一座有些奇特的坟墓。土坛上的网形图案也许什么用心也没有，而是像汉代出土的砖上有绳纹一样，只起一种装饰作用。王康想让工匠们把这个安放爱女棺木的土坛弄得好看漂亮些，匠人们就弄成了这个模样，结果害得我们这些后人百思不得其解…… <br/>　　“嗨，你是来打工摘茧的吗？”一声问话把卓月的沉思打断，她扭头才发现身后站着一个手提盛茧竹筐的小伙。 <br/>　　“你要是来摘茧挣钱的，请到岗尾的蚕茧基地去登记一下，领一个竹筐，然后就可以满岗上找柞树枝子摘了，基地按你摘茧的重量付钱！” <br/>　　“不，”卓月摇摇头，“我不打工，我是来看这个的！”她指了一下那个土坛。 <br/>　　“这个土场子有啥看头？它最早还是我妹妹发现的！” <br/>　　“你妹妹？”卓月惊异了，“你是宁贞的哥哥？” <br/>　　“对呀，我是她哥哥曹宁安，你认识她？” <br/>　　卓月点点头，一边打量着宁安一边想起宁贞当初为哥哥卖假酒事急得抹泪的样子。 <br/>　　“你说这土场子上用颜料画出的方格子是啥意思？” <br/>　　“你说呢？”卓月笑了。 <br/>　　“左大哥说它是在暗示一种东西。” <br/>　　“左大哥？你左大哥是干什么的？叫什么名字？”卓月惊奇了。 <br/>　　“他叫左居士。他懂的事情可多了。” <br/>　　“他在哪儿？”卓月来了兴趣。一个人能指出这土坛上的图案是在暗示一种东西，表明他起码是一个有思想的人。 <br/>　　“他就在这蚕茧基地打工，你等一下，我喊一声看他能不能听见。”宁安说罢，放开嗓子朝岗上喊了一腔：“左大哥——” <br/>　　没有回音。 <br/>　　“不晓得他今儿个到哪条岗上摘茧了。”宁安摊摊手。 <br/>　　“你那位左大哥没有说这土场子上的方格子是在暗示什么吗？” <br/>　　“说了，我不大懂，也记不清。” <br/>　　卓月那天下岗时想到，以后应该找机会来见见这位姓左的，民间也常常隐藏着一些能人和智者，说不定他对这网形图案有另一番解释，可以帮助自己打开思路。暗示？我过去一直在琢磨它是什么含义，它代表一种什么东西，还很少想到它在暗示，暗示是一种带有动感的东西。这种解释倒有点意思。 <br/>　　它在暗示什么？它在暗暗地给后人提示什么东西？ <br/>　　左大哥？左居士，倒是和左涛同姓。世上还有名字就叫居士的？居士不是佛家对一种信徒的称谓吗？…… <br/>　　经过几天的琢磨，卓月意识到，如果把方形土坛只看做一座有些奇特的坟墓，那土坛上用颜料画出的网形图案，就可以说成是起一种对死者宽慰、安慰的作用；但如果把网形图案说成是一种暗示的手段，则土坛上的棺材和尸骨就会具有另外的意义。意识到这个问题之后，她决定对蚕茧基地的左居士作一次拜访，听听民间对方形土坛和土坛上那个图案的看法。她那天下午专门去了一趟尚吉利丝织厂，让宁贞转告她哥哥宁安，要宁安告诉那位左居士，说她第二天上午要去拜会左居士，让他第二天上午在蚕茧基地的茧库门口等她一会儿，她九点钟准时到。宁贞当时脆脆地应了一声：“没问题。”第二天早上来上班时，她还专门骑车过来告诉卓月，说她已经给她哥哥说了。 <br/>　　卓月骑自行车于上午九点准时赶到蚕茧基地的茧库门口，但门口却只有宁安一个人站在那里。宁安看见卓月急忙过来解释：“我给左居士说了你要来见他，并要他等在这儿，这会儿却不知他跑到了哪里，他这人脾气古怪，平日不愿见人绝少说话，兴许听说你是师大的副教授，更有些害怕了。你先在这儿等，我去找他！” <br/>　　卓月点头笑笑说：“麻烦你了。” <br/>　　但宁安找了半天也没找到。卓月直等到中午，还不见对方的影子，只好对宁安说：“算了，我以后再来吧。” <br/>　　她悻悻地骑上自行车往回走。她的涵养一向很好，可此时也忍不住在心上抱怨：左居士，我既是提前约了你，你不该让我补空的！…… <br/>　　第二天早上，卓月正做早饭，宁贞来了。宁贞进屋就说：“我听说左居士昨天没见你，昨晚我专门找到他抱怨他了一顿，他说他一个打工的百姓，不想和外人打交道，所以昨儿个就躲开了。” <br/>　　卓月笑笑：“我一个教书的见他，还能把他吃了？” <br/>　　“也是，”宁贞也笑道，“他这人脾气有些古怪，昨晚我临离开他时，他让我转告你，说请你不要去找他了，说关于方形土坛上那个用颜料画出的图案，他可以把他的理解告诉你。他说他认为那个图案是在暗示：人不仅仅要看到在场的东西，还要看到不在场的东西，不在场的东西并不等于‘无’。他说的话非常难懂，我不知道我转述得是否正确。” <br/>　　“哦？”卓月的眼瞪大了，宁贞转述的这几句话让她一愣，这些话带有一种明显的哲学思考的意味，莫非那个姓左的真是个民间智者？ <br/>　　“他说，那个图案中的线条，是在提醒观察者注意在场的东西；那个图案中的空白格子，是在提醒观察者注意不在场的东西。” <br/>　　“他还说了什么？”卓月的语气有些迫不及待了。 <br/>　　“他说，那个图案是古人精心设计的，不是一种随意行为，不能看成一种装饰图案。”宁贞皱了眉努力回忆着。 <br/>　　“还说了什么？” <br/>　　“没有了，好像就这些。”宁贞摇摇头，吁一口气。 <br/>　　“好，谢谢你！”卓月高兴地拍着宁贞的肩膀。送走宁贞之后，卓月坐在书桌前，长久地望着自己当初在纸上画成的那个方形土坛和土坛上的网形图案，在心里叫道：有意思！这个姓左的说得有意思！那些线条是在提醒后人注意在场的东西，也就是棺材、尸体、陶器、环首刀、牛和羊的尸骨？那些空白的格子是在提醒后人注意不在场的东西，也就是导致这些尸体存在的人物、事件、缘由？如果照这样理解，那整个网形图案就是在进行一种暗示。 <br/>　　这倒是一种新见解！ <br/>　　在场的东西和不在场的东西以及“无”，构成了这个世界？</p></div>
