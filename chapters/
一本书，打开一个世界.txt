<h1 id="chapter_title_177988"><strong>一本书，打开一个世界</strong></h1>
<p class="info">
分类：<a class="cate" href="/novel-list-52.html" target="blank">散文</a><span class="divide">|</span>作者：<a class="name" href="/pdnovel.php?mod=search&amp;ac=author&amp;srchtxt=无名&amp;searchsubmit=yes"  target="_blank">无名</a><span class="divide">|</span>发布时间：2014-08-22 08:56:03<span class="divide">|</span></p>
<div class="page-body">
<div class="page-content">
<p>世界上总共只有一本书。学富五车也好，汗牛充栋也好，林林总总的方册只不过是一本书的分章继节而已。 <br/>　　这本书，不管用什么书写——是行云流水、是雪泥鸿爪、是虫蛇行迹、是捉刀刻形，更不必计算用了多少字写成的，写出的意思只念一个音——人。其他的字符皆是人的笔画。 <br/>　　书者，大乎哉！一本书立起来是座山，打开来是片海，放下来是座人生舞台。 <br/>　　书的山，不是石的山。石的山：“山不过来，我过去。”书的山：我不过去，“山”过来。山用霞光过来；山用影子过来；山用飞鸟过来；山用飞瀑过来；山用松涛过来；山用花香过来；山用罡风过来……山是可翻越的山，山是能走进去的山。山中有风雨，书中亦有风雨——书中的风雨是人遮挡过的风雨，所以汗青是干燥的、心画是完整的。山中有冰雪，书中亦有冰雪——书中的冰雪是人踏卧过的冰雪，原来坟典中的冷暖正是人的体温。读着，走着，你或许成了书中的主人公；你或许成了书中的某个角色。留心看，准有几页预写了你的故事。不用摸，准有一空一格预设了你的位置。累了，你可以在书中找个太阳能晒出温暖来的词，坐下歇歇，不一定是把椅子，转椅更不牢靠的，忽忽悠悠不说，有时就找不着北。掀起的这一页是天，按住的这一章是地，天算的页码，地排的顺序，早读是多少，晚读是少多。别急，你放松放松，你斜倚在一块为你铺上了青苔的陋石之侧，点地上的蚁穴星星；你仰卧在一席为你点缀了黄花的芳草之上，数天上的白云朵朵。渴了，你掬一捧流淌在字里行间的清泉，撷两滴盘挂在耳畔的鸟鸣……定睛看哪！花丛中兀自涌出一双明眸秋波……哦，你醉了……饿了，书中有你打打牙祭的食物，掀开最温暖的一页，你便揭开了锅盖，端出母亲为你留的饭菜。一袭芳香引领你踏进了一片果园，你喊一声：“有人吗？”“扑楞楞”——惊飞了枝头那儿正在品咂禁果的两只喜鹊…… 读着，走着，有人一不小心于书中迷失了自己。读着，走着，你在书中逮住了你的童年、撞疼了你的青年，绊倒了你的中年——你已看见了你老年蹒跚的背影。你挣破了不是丝的不是纸的包装，从一本书里挤出来，你成了一页新的你。<br/>　　书的海，不蒸腾的水。书的海向你漫了过来，湿了，湿了的是人之外的衣服，全身心地浸入书之水吧，水中，衣服是多余的。水载你而起，你是会水者。此时，文字们都成了伴你而游的鱼。当弄潮儿太累，“胜似闲庭信步”蛮好。海会无风三尺浪，文字的鱼们纷纷张开有的齿没的齿，咬你！咬你！咬你！哎呀！那边追上了海明威的鱼，吃光了海明威的鱼，眼看着就要追上海明威的胡子了，没法子，海明威朝自己嘴里的海，开了一枪…… <br/>　　书的舞台，是人装订的平台。书的平台就在你的脚下，你不表演行吗？不表演你就下不来台——叮当作响的台；红毡铺地的台；打着灯光变着色，粘上布景换着彩的台；坑坑包包的台；布满了陷阱的台；温柔之乡的台；“傻小子睡凉炕——全凭火力壮”的台；颠簸你的台；向上抬你的台；往下出溜你的台。书的台，四四方方的台——东西南北中的台；一不小心闪了脚就会掉下去的台；有边有沿，有楞有角的台；一扭、二晃、三摆，走到头了的台——还想往前走就得继续支撑的台。 <br/>　　为人者不可不读书。知书才化冥顽；知书才能达理；知书才通人性。有人读书读好了；有人读书读坏了；有人读书读精了；有人读书读傻了；有人读书读活了；有人读书读死了。 <br/>　　一本书，不同的人读，便有不同的收获，所谓“仁者见仁，智者见智”。一部《红楼梦》——“道家看法；儒家说仁；佛家悟惮；闺阁中人体会言情”。一本书在不同的地点读：枕上读、厕上读、马上读、车上读、兰前读、松下读、山上读、海边读，各得异样味道，各享不同境界。一本书不同时间看，会有不同的效果。小时候看热闹；长大后看门道。人说，少不读《水浒》，是怕年青人血气方刚帽子压不住冲破脑门去惹事、去造反——“杀去东京夺了鸟位”；人说，老不读《三国》，是劝人老了，该安静了。别再多了诸葛亮的鹅翎扇，多了曹操的花白脸，多了司马懿的小心眼——怎就多了尔虞我诈，多了鼓角铮鸣刀光剑影…… <br/>　　 “一花一世界，一叶一乾坤”不大，更大是一页百乾坤，一册千世界。 <br/>　　一本书浪漫了一个世界——一个“思无邪”的世界，如《诗经》——“关关雎鸠”还在啼鸣，而“在河之洲”上的脚印，乱了。 <br/>　　一本书算计了一个世界，如《周易》——世界是由两条阴阳鱼加上18道长长短短、短短长长的杠杠组成的。八卦相加，究竟得多少？ <br/>　　一本书智慧了一个世界，如《论语》——人说是仁义点好，嘴唇是仁义的，可嘴唇掩护的牙齿却是锋利的；人说是聪明了乖，手，听脑袋指挥，可是手延长的部分——手指甲却不满足知觉……人说是“半部《论语》治天下”，那半部呢？那半部，大概是没读懂，只好用一半；或许是读懂了，故意用一半。人，活在这个世界上，一半聪明，一半糊涂才是“大聪明”。 <br/>　　一本书打赢了一个世界，如《孙子兵法》——“兵者，国之大事。死生之地，存亡之道，不可不察也”。“兵者”，战场也，情场也，商场也。战场上，人们要死要活地察，一响之后，人们手里握硬的只有一块石头了。情场上，人们又爱又恨地察，一夜之后，人们手里握热的只留一个地址了。商场上，人们挖空心思地察，一刷之后，人们手里握扁的只剩一张卡了。 <br/>　　一本书仰止了一个世界，如《圣经》——上帝只有一个，这个上帝的架子太大，书搭的架子，十字的架子，活着的人谁见过架子以下的上帝？休怪、休怪，一个上帝帮不过来这么多的人，究竟谁帮谁了？谁都“谢谢上帝”；谁都“阿门”出口。 <br/>　　一本书里隐藏了一个世界，如《伊索寓言》——动物的世界里，分明看见了人在张牙舞爪。 <br/>　　一本书小了一个世界，如《安徒生童话》——人，长不大，多好。 <br/>　　一本书忧郁了一个世界，如《少年维特之烦恼》——少年烦恼了，一蹦达就过去了；老了烦恼了，可就要了命了。 <br/>　　一本书硬是将世界分成了两种颜色，如《红与黑》——是红的贵？还是黑的贱？有价格的故事是用黑字印成的。黑字印的书，人，瞅着眼睛不发红。红，不出血，咋个红法？黑，只晓得拉上红窗帘，屋子里，黑，黑了…… <br/>　　一本书敲响一个人，一个人用钟声超越了一个尖顶拱门里的世界，如《巴黎圣母院》——背，是驼的，一个世界压驼的，钟声是直的；脸，是丑的，一个世界照丑的，钟声是美的。爱是又直又美的。爱到只写下两具相拥的骸骨时，还须辨认美和丑吗？爱到一起化作尘烟时，尘烟也是又直又美的。 <br/>　　一本书残害了一个世界，如希特勒的《我的奋斗》——一个留着小黑胡的人疯了，一个东一块、西一块的世界疯了……子弹头做笔，写出的是血、是仇恨、是耻辱。化剑为犁固然是好。无奈，德意志的剑，横行霸道也犁不出“第三帝国”的梦——太阳下山了，还是回到自家的地里挖面包去吧。 <br/>　　一本书安静了一个世界，如肖洛霍夫的《静静的顿河》——顿河静了，白桦树静了，草原静了，哥萨特的小白屋，静了。可是，飘荡的旗帜风，不静，马刀挥舞的风，不静…… <br/>　　一本书金属了一个世界，如奥斯特洛夫斯基的《钢铁是怎样炼成的》——炼成的钢铁打造了镰刀与锤子，炼成的钢铁打造了一代人的楷模。 <br/>　　一本书推动了一个世界，如爱因斯坦的《相对论》——世界是圆的；原子弹也是圆的，原子弹与世界相对，将会怎么样？呜呼！ <br/>　　一个人用一本书将人的魂儿领入了一个鬼狐的世界，如蒲松龄。《聊斋》的世界，盛在了十字路口大槐树下的茶碗里；撂在了荒郊野外的孤坟荒冢里；研在了狼毫渴舔的端砚里；飘在了红袖添香的清风里。《聊斋》的世界确实有，但不是每个人都能抵达。只有比人干净——赶得上狐狸那样干净的人，才有此艳遇，才有此幸福，才有此逍遥。<br/><br/>一个人用一本书诠释了整个世界，如巴尔扎克。《人间喜剧》150年前上演过了，如今还在上演，以后还会继续演演演…… <br/>　　一个人用一本书将人孵成了虫子，如卡夫卡。《变形记》读出多少形，人就能变出多少形。虫子比人能伸展得开，不像人，做事时攥着拳头，说话时卷着舌头；虫子比人自由自在，吃饱了就睡，没趣了就飞。在虫子眼里，人，是个什么东西？ <br/>　　有的人一辈子没写完一本书，如曹雪芹。曹雪芹用三分之二的《红楼梦》演绎了整个世界。曹雪芹的《红楼梦》，是一个水做的——女人的美丽世界；是一个初醒的、有点“狂”——一个“新人”胚胎的男人，“爱博而心劳”地与一群女人“昵而敬之”的世界；是一群女人活得有点像人了的世界；是“只有门前的那对石狮子是干净”的世界；是“……将有价值的东西毁灭了给人看”的世界；是曹雪芹在梦里才能写出的真实世界。一个世界在一个梦的预言中訇然倒塌了，遗憾的是，睡在中国人骨子里的《石头记》的残渣，还疙疙瘩瘩、硌硌楞楞的没有完全醒来。 <br/>　　有的人主张一辈子只写一本书，如丁玲。《太阳照在桑干河上》了，太阳一照，河水都亮堂了，还有什么不明白？潺潺流水日夜叙说，还用人再唠叨些什么？ <br/>　　一个人想用一本书拯救一个民族，如鲁迅。《狂人日记》里所记的正是正常人的呻吟；《阿Q正传》传下来的小辫儿，甩甩达达地成了一个民族发了炎的阑尾，《孔已己》的长衫被撕成了包扎昨日伤口的绷带，血浸的馒头，成了治痨的《药》，中国病的不轻呵！鲁迅一劲儿地咳嗽。鲁迅将中国的黑夜咳出了一个洞——一个透进了光明的洞，以至于今天，我们一读鲁迅还是气喘吁吁的。 <br/>　　一个人用一本书打败了另一个人，如毛泽东。蛰居台湾的蒋中正通读了《毛泽东选集》后，掩卷一声长叹：吾不如毛润之也。服了。 <br/>　　一页页看书，一本本看书，看了一本书结识了一个人，这个人有可能是自己。看了一本书认识了一群人——一群熟悉的人；一群陌生的人；几位英雄；仨俩小丑。 <br/>　　有的人，一本书是他的功德碑；有的人，一本书是他的遮羞布；有的人，一本书是他的敲门砖。有的人因为一本书而招来没顶之灾，老舍是也；有的人因为一本书而名垂千古，老舍还是也。 <br/>　　没在意人人读书，却一定人人在写书——一行脚印，一行文字。你刚写完开头，你已经进入中段；你紧张着哪，你正在“刘郎已恨蓬山远，更隔蓬山一万重”的高潮之上；你浪漫着呢，你正在“不信楼头杨柳月，玉人歌舞未曾归”的陶醉之中，你得意着哪，你正在“气蒸云梦泽，波撼岳阳城”的成功之时，你懊丧着呢，“志士嗟日短，愁人知夜长”，你昨晚经历了失败…… “只有一枝梧叶，不知多少秋声？”谁能预先写完自己结尾的，谁是先知先觉的伟人哪！ <br/>　　自己写的书自己能读懂的，是明智之人；自己写的书自己看不明白，定是混帐之人——你糊弄了自个儿，却糊弄不了别人。 <br/>　　一个人的一辈子被另一个人写成了一本书；一个人将自己的一辈子写成了另一个人。 <br/>　　一本书可以改变一个人：一个人一辈子快乐之书；一个人一辈子痛苦之书；一个人一辈子郁闷之书；一个人一辈子彷徨之书；一个人一辈子呐喊之书。 <br/>　　一个人读完一本书，不难；一个人读懂了一本书，难；一个人重读一本书，有点难；一个人重读一本书时幡然读出了新意，难上加难。
