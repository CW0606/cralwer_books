<h1 id="chapter_title_177988"><strong>第二部 第1节</strong></h1>
<p class="info">
分类：<a class="cate" href="/novel-list-23.html" target="blank">文学小说</a><span class="divide">|</span>作者：<a class="name" href="/pdnovel.php?mod=search&amp;ac=author&amp;srchtxt=周大新&amp;searchsubmit=yes"  target="_blank">周大新</a><span class="divide">|</span>发布时间：2015-04-24 23:46:23<span class="divide">|</span></p>
<div class="page-body">
<div class="page-content">
<p>值夜的还没有敲响五更的梆子，晨曦还未洇进窗纸，尚达志就拍醒了睡在身边的九岁的儿子尚立世：“穿衣吧，小世！”说着，自己先穿起衣来。尚未完全脱离睡乡的小立世在被窝里含混地哼了两声，刚要俯身再睡，达志却已呼地掀开了整个被子，把光赤着身子的小世儿晾在了床上。早晨的寒冷倏地扑来抓挠小立世那柔嫩的肌肤，赶走了他的最后一点睡意，他这才一骨碌坐起身，急忙去抓衣服。 <br/>　　“小心凉着。”睡在另一张床上的立世娘顺儿，这时扯下女儿捏着她奶头的小手，穿着内衣跟着鞋跑过来，急忙帮儿子去穿裤子。 <br/>　　“我在桑园等你！”尚达志朝儿子丢下一句，便拉开门走了出去。 <br/>　　早春的晨风还带有挺利的爪子，拂过脸时还让人觉出有些疼痛，小立世挟好书本走出门没有几步，便急忙抬手去抚了一下脸颊，哈出一口气，但他没有停步，他知道爹的脾气，他必须立刻赶到桑园晨读。他那细瘦的双腿在晨光里快速地摆动，跑到后院古桑树下时，他已经喘成了一团，小小的胸脯急剧地起伏着。 <br/>　　“立世，昨日早上我给你说过，要织出‘霸王绸’须得做到四戒，还记得四戒是啥吗？”待立世的喘息稍一平下，达志立刻就开了口问。 <br/>　　“记得，四戒是戒酒、戒赌、戒淫、戒鸦片！”小立世站直了瘦小的身躯，脆声答。这孩子大约是因为顺儿身体瘦小先天给他的营养不足的缘故，发育得慢，显得十分清瘦。 <br/>　　“为啥要戒酒？” <br/>　　“酒能醉人，让人忘了正事；酒能废人，使人智力消退！”小立世看来已记到了心里，答得滚瓜溜熟。 <br/>　　“为啥戒赌？” <br/>　　“天下之败家者，多迷于赌。赌可以使人精神恍惚，琴书都罢，田园尽弃，卖儿鬻女！” <br/>　　“为啥戒淫？” <br/>　　“生为男人，当做经天纬地事业，若沉淫欲之中，轻则损精费神，未老而衰；重则元阳丧失，业废嗣绝！” <br/>　　“为啥戒鸦片？” <br/>　　“食鸦片者，肩耸项缩，颜色枯羸，家资耗尽，死期亦至！” <br/>　　“嗯。”尚达志满意地点了下头，“好了，读你的书吧！” <br/>　　前院传来织机的响声，小立世知道那是娘和雇来的两个女工已经上机。咔咔咔，他就在这有节奏的响声里全神阅读。尚达志便也在这响声里默想着自己复兴尚吉利大机房的计划。 <br/>　　九年了，凭着顺儿和自己的一双手，倒闭了的机房总算又活了过来。如今，每天可以有三台织机干活，其中，一台是由顺儿包着，另两台是由雇来的两名女工操作。规模虽然不大，离父亲和爷爷的设想差得太远，但就这样一个局面，达志是凭了怎样的努力才实现的呵！当初尚安业死后，达志手里有的就是那十几两银子，可那怎够重新开业？达志是靠自己外出给人修织机、靠顺儿织卖土布逐渐积攒了点钱，尔后又把放在几家机户里的织机卖了，这才又开始买点丝织绸。先上来本钱太小，不敢雇人，就顺儿一人织，待积了点钱后，才又相继雇了两个女工，把规模扩大到今天的样子。 <br/>　　必须买机动织机！达志的拳头在面前的老桑树上砸了一下。如今因为产量太少，尚吉利机房的产品根本引不起外地丝绸商人重视，基本上没有什么竞争力，而要提高产量，仍靠木质土造人工织机，需雇的人多，成本也高。可要买机动织机，钱在哪里？积这么几年，达志手上也只有百多两现银。 <br/>　　唉，银子呀！ <br/>　　“爹，第六章我读完了。” <br/>　　小立世的声音把达志从苦思苦想中拽出，他看了一眼儿子扬起的书本，点点头说：“好，现在开始背那三段话吧！” <br/>　　“自唐武德八年始，吾南阳尚家从丝绸织造，迄今已千二百八十五年，绩煌煌。”小立世流利地背道，“北宋开宝二年，吾尚家所出之八丝绸，质极好，被中外绸商誉为‘霸王绸’，所产之大部，贡皇室；亦有一部售西域……” </p></div>
