<h1 id="chapter_title_177988"><strong>（三十七）</strong></h1>
<p class="info">
分类：<a class="cate" href="/novel-list-23.html" target="blank">文学小说</a><span class="divide">|</span>作者：<a class="name" href="/pdnovel.php?mod=search&amp;ac=author&amp;srchtxt=魏巍&amp;searchsubmit=yes"  target="_blank">魏巍</a><span class="divide">|</span>发布时间：2015-04-23 23:31:52<span class="divide">|</span></p>
<div class="page-body">

<p>金沙江在黑沉沉的夜里发出沉重的涛声。 <br/>　　在离江边不远的一座小村里，一家农舍小小的窗户上还亮着灯光。 <br/>　　这是临近大路的一家小店。店主东又兼店小二的张福，正赤着膊佝偻着身子躺在他那肮脏的床铺上抽大烟，只听有人卜卜卜地叩击着窗棂，随后轻声唤道： <br/>　　“老板，快开门罗，你不要怕，我们是红军，是打富救贫的！” <br/>　　张福一听是红军，愣住了，眼睛盯着窗户，拿着烟葫芦的手索索发抖。这几天都在传说要来红军，谁知道红军怎么样呢？再说，人讲红军还在二百里以外，怎么眨眼已经到了？ <br/>　　窗棂又卜卜地响起来，还是那样轻声地呼唤： <br/>　　“老板，你不要怕，我们是红军，是打富救贫的！” <br/>　　“打富救贫”是红军经常使用的一个通俗口号。尽管这口号不甚科学，但它一听就懂，能很快为贫苦人所理解，所接受。张福第二次听到它时，心就有些动了。等到那轻轻的呼唤声再次送到耳边，他就放下大烟葫芦，下地开了门。 <br/>　　首先进来的，是一个面孔白皙、英姿勃勃的青年人。他身穿灰色军服，头戴红星军帽，腰插驳壳枪，象是个军官的样子。其余的人都留在门外，有带短枪的，有带长枪的，有穿军衣的，有穿便衣的，夜色朦胧，一下也看不清楚。 <br/>　　那个脸孔白皙的青年人，见张福仍有些胆怯，就和颜悦色地说： <br/>　　“老板，打搅你了。这里离皎平渡远吗？” <br/>　　张福见这青年人十分和善，听声音刚才叫门的也想必是他，心慢慢定了下来，就连忙答道： <br/>　　“不远，下去就是。” <br/>　　“有没有船？” <br/>　　“船，倒是有两条，都是金保长家的。可不晓得还在不在。” <br/>　　“船在哪里？” <br/>　　“听说今天一早，区公所给他来了一封木炭鸡毛信，叫他把船烧了。” <br/>　　那青年红军一听急了，忙问： <br/>　　“船烧了吗？” <br/>　　“不晓得。” <br/>　　青年军官打量了一下这个破破烂烂的屋子，拍了拍张福的赤膊，充满热诚地说道： <br/>　　“老板，看样子你也不算很宽裕吧。” <br/>　　张福心里一酸，苦笑着说： <br/>　　“我原本也是个船夫，后来叫人解雇了，没得办法，开了这个小店混碗饭吃。” <br/>　　“你帮我们带带路，找找船行吗？” <br/>　　“行，行。” <br/>　　青年军官见张福答应得很爽快，很是高兴，立刻同张福一起走出门外。 <br/>　　这地方白天很热，晚上阵阵江风吹来，倒颇有些清冷。青年军官见张福还打着赤膊，就从背包里抻出一件旧衣服给张福披上，张福推辞了一番，才舒上袖子，心里不禁热烘烘的。 <br/>　　青年军官一路走，一路探问对岸的敌情。张福告诉他，对岸通安县驻着川军一个团，渡口上的敌人倒不多，只有保安队五六十人。另外还有一个专门收税的厘金局，有两名武装保丁。从谈话中已经可以听出这个店主东很亲热了。 <br/>　　谈话间已经来到江边。对岸什么也看不见，只有几点稀疏的灯火。高耸的山岭，在夜空里象炭块一般地画着粗犷的弧线。滔滔的江水模糊一片，显得幽深可怕，只能听见呜呜的流水声震人心魂。 <br/>　　青年军官带着侦察队来到渡口，反复细看，连船的影子也没有。张福也显得犹豫不定。这时，忽然听见岸边石头上仿佛有人低声说话。走上前一看，原来是金保长家的船工张潮满和他的十五六岁的儿子大潮正坐在石头上闲话。这张潮满将近五十年纪，最近老伴死了，儿子给金保长家放马，因为顶撞了东家几句，被辞退了，心中甚为抑郁烦闷，来到江边闲坐。张福见他父子有些惊慌，就低声说： <br/>　　“潮满哥，你别害怕，他们是红军，是打富救贫的。他们要过江，你知道船在哪里？” <br/>　　张潮满沉吟了一下，说： <br/>　　“今天高头来命令让烧船，金保长不舍得烧，把那条新船开到江那边去了。” <br/>　　“那条旧船呢？” <br/>　　“旧船，已经废了，藏在李家屋头那个湾湾里，进了半船水了。” <br/>　　青年军官立刻插上来说： <br/>　　“老大伯，你能带我们去看看吗？” <br/>　　张潮满犹豫了，在黑影里没有作声。 <br/>　　张福插上来说： <br/>　　“潮满哥，你就带他们去一趟吧！” <br/>　　“不是我不愿去，”张潮满嗫嚅着说，“要是让金保长知道……” <br/>　　张福立刻说： <br/>　　“不要紧，天塌下来大家顶着，再说红军也是为了我们干人！” <br/>　　“阿爸，去吧，怎么也好不了。”大潮也说。 <br/>　　老人站起来，慢慢地移动着脚步。 <br/>　　众人跟着走了不远，在满是林莽的一个小岔子里，找到那只旧船，果然进了大半船水。 <br/>　　张福在附近人家找了几个水桶盆罐，大家就跳到船里淘起水来。这时老人又教训自己的儿子： <br/>　　“你愣着干什么，还不快下来淘水！” <br/>　　不一时，淘净了水，就把那只船拉到岸上。青年军官检查了一遍，反而叹了口气： <br/>　　“这条船破成这个样子怎么能开？” <br/>　　张福笑着说： <br/>　　“你不要急，我有办法。” <br/>　　说着，他一溜烟跑回他的小店，不一时抱了几床破被子来，说： <br/>　　“大家把它扯成布条条，把缝子塞住，也许能行。” <br/>　　青年军官异常高兴，从腰里摸出十几块银元，塞给张福。 <br/>　　张福推辞不要，青年军官说： <br/>　　“这怎么行，我们红军不拿老百姓的一针一线，你这是多少条线哪！” <br/>　　张福也就笑着收了。 <br/>　　大家立刻动起手来，把破被子撕成条条，用刺刀把布条堵在船缝里。收拾完毕，推到水里一试，果然没有进水。大家十分欢喜。张福就跳上船去，同张潮满父子一起，把船划到渡口。 <br/>　　这时，后续部队陆续来到，渡口上黑鸦鸦的，总有一百多人。张福看见青年军官跑上去报告，夜色里传来一阵满意的笑声，接着，一个爽朗的声音说： <br/>　　“肖队长！要快！你们马上渡江，抢占渡口！” <br/>　　张福才知道这个青年军官叫“肖队长”。肖队长立刻招呼大家上船，除了他的队员，又上来了一个排。一切安排妥当，就叫张福和张氏父子开船。 <br/>　　木船划入黑魆魆的江水中，激流冲着船体嘭嘭作响。船不断地一浮一沉，不时有浪花打进船内，战士们怕打湿枪枝，把枪枝紧紧地抱在怀里。肖队长却高高地昂起头，聚精会神地疑视着对岸的灯光。 <br/>　　此处江面甚宽，划了很长时间，才越过中流，渐渐靠近对岸。张福悄悄地向上一指，肖队长看见靠岸处向上是一排石级，最上面站着一个哨兵。他对两个战士耳语了几句，两个战士就跳下船去，向那个哨兵悄悄接近。不一时就听上面那个哨兵厉声问道：“干什么的？”两个战士不慌不忙地回答：“自己人！”接着就扑上去了。没有费多大事这个哨兵就当了俘虏。 <br/>　　大家下了船。肖队长叫那个排带上俘虏去右面解决那个新开来的连队，自己就同张福一起到厘金局来。 <br/>　　走了不远，就来到厘金局。张福指着一个门，轻声说：“这里就是。里面有个姓林的克扣穷人，可坏透了！”肖队长附在他的耳边悄声地说： <br/>　　“我们口音不对，还是你来叫门的好。” <br/>　　张福点点头，就开始敲门，一面温声细语地叫： <br/>　　“林师爷！快开门，我们是来上税的！” <br/>　　里面一个粗哑的声音厌烦地说： <br/>　　“天还不亮，不办公事。” <br/>　　张福又带些哀求的口吻叫： <br/>　　“林师爷，我们是赶猪的，猪已经赶到沙坝来了，天一亮，我们还要到绞西买猪料呢！” <br/>　　里面又粗声粗气地说： <br/>　　“我已经说过了，天亮再说！” <br/>　　张福向肖队长挤挤眼，大声叹了口气，说： <br/>　　“是这样，林师爷，我们还要赶路，要不我们只有把猪赶到昆明去卖，那只好下一次再到你这里上税了。” <br/>　　这一下果然很灵，里面咳嗽了两声，接着点上灯，开了门。肖队长见里面一个满面烟容、瘦脸长眉的老家伙，披着衣服站在那里。他用手枪一比，说： <br/>　　“我们是红军，快把枪交出来！” <br/>　　那位在乡下人面前一向两眼望天的林师爷，立时吓得面如土色，全身筛起糠来。他冲着里间屋颤抖着说： <br/>　　“快，快，你们快把枪扔出来！” <br/>　　两枝步枪从里间屋里扔出来了，接着出来了两个保丁。 <br/>　　人们拿着缴获的枪枝，回到渡口。这时，右边那个排，也进展得十分顺利。原来他们由俘虏带路，很快就闯进了江防连的驻地，不费一枪一弹就俘虏了几十个人，因为他们正美滋滋地在吞云吐雾呢！ <br/>　　江滩上烧起很大一堆火来，这是向对岸发出的占领渡口的信号。那火焰在夜空里欢跃地抖动着，江水也反射着一片抖动的红光。在火堆旁边，肖队长那张白皙英俊的脸闪着光彩，张福正对着他嘻嘻地笑。 </p></div>
