<h1 id="chapter_title_177988"><strong>第五卷 干 第九章 絮言——天堂日子</strong></h1>
<p class="info">
分类：<a class="cate" href="/novel-list-23.html" target="blank">文学小说</a><span class="divide">|</span>作者：<a class="name" href="/pdnovel.php?mod=search&amp;ac=author&amp;srchtxt=阎连科&amp;searchsubmit=yes"  target="_blank">阎连科</a><span class="divide">|</span>发布时间：2015-05-23 15:56:27<span class="divide">|</span></p>
<div class="page-body">
<div class="page-content">
<p>①洋车子：即自行车，在豫西耙耧山脉，最早称自行车为洋车子，后为脚踏车，再后多年实行破四旧，让耙耧百姓嘴里不能说洋字，才改称为了自行车。但时至今日，那里的老人还有说自行车是脚踏车和洋车子的。　　<br/>　　③天堂日子：天堂日子是庚寅年秋受活人成立了互助组后的一段异常特殊的集体主义的劳动生活。　　<br/>　　各家的田地都合到了一块，牛和犁、耧、锄、耙都充了公。那些有牛、有犁、有车的明显吃了亏，原是想哭想闹的，可又有几声枪响后，他们就不哭不闹，交了牛、车和犁、耙。　　<br/>　　横竖互助组是成立起来了。区长和民兵在庄里住了三天，把扛来的枪带走了一支，另一支就留到庄里了。　　<br/>　　留给了茅枝。　　<br/>　　原来茅枝曾是队伍上的人，是打过了仗的人，其经历比区长还老，和县长齐肩。　　<br/>　　原来，她自小就是革命者。就是执政者。　　<br/>　　接下来，在村中央的树上挂了牛车轮子钟，茅枝一敲，受活人就都集合着下了田。她说到东山去锄地，受活人就都到东山去锄地，她说到西山去施肥，就都到西山去施肥。原来，互助组竟是那样好，千百年来受活都是各家种着各家的地，你犁他播，一家在山顶，一家在沟底，大小事都需扯着嗓子吼。瘸子家要借用聋子家的土箩筐，那唤没有用，就要从沟底一跛一跛爬到梁顶上，再一跛一跛走下来。可到了互助组，这些都不再需要了。茅枝敲了钟，唤着说都扛铁锨啊——你扛着铁锨下地就完了。唤着都挑上箩筐啊，你挑上箩筐就行了。　　<br/>　　下地的路上，爱说话的人，就不再寂寞了，不爱说话的人，耳朵也不寂寞了。　　<br/>　　收工回来，你爱唱耙耧调、祥符调、曲剧或梆子，那你就扯着你的嗓子唱，你不会因了没人听戏就冤了自己的才华和嗓子。　　<br/>　　冬天过去，春日走来，一敲钟，就让男女老少，除了瞎子和瘫子，其余别的都下田锄小麦。先锄村东最大的一块地，十几亩，斜斜地挂在山坡上，像是掉在山坡上的一块儿天。男女老少，瘸的拐的，聋子哑巴，能扶锄的都并着肩膀锄，拢共数十人，一行儿排开，起锄落锄，黄白亮亮的嚓嚓声响满了山梁子。　　<br/>　　有一个瘫媳妇，她不能站立，自然也不能去扶锄锄地，茅枝就让大伙锄地时她在田头上唱戏给大伙听。还有一个实瞎子，自小不知道天是啥颜色，地是啥颜色。可他自小爱听人唱戏，听了也就会唱了。茅枝便也让他来和瘫媳妇一块唱。　　<br/>　　村人们锄着地。他们唱着戏。他们唱祥符调里的《双玉燕》《蝴蝶传》，唱耙耧调里的《响马传》《二女多情传》，到了没有戏词时，就随口编了一曲《我没有老婆你没有汉》——　　<br/>　　男瞎子唱：　　<br/>　　场上麦子堆成二十一垛　　<br/>　　谁想到哥哥我没老婆　　<br/>　　独头蒜儿不分瓣　　<br/>　　可怜哥哥我光棍汉　　<br/>　　女瘫子唱：　　<br/>　　耙耧的风箱两头拉　　<br/>　　啥儿人留下我守寡　　<br/>　　前辕骡子后辕儿马　　<br/>　　谁知我妹妹守了寡　　<br/>　　男瞎子唱：　　<br/>　　没老婆的哥哥没笼头的马　　<br/>　　日落西山哪是哥哥的家　　<br/>　　日头落在西山沟　　<br/>　　没老婆的哥哥谁收留　　<br/>　　女瘫子唱：　　<br/>　　炉子里冒烟笤帚扇　　<br/>　　守寡的妹子我孤单单　　<br/>　　月亮上来明晃晃　　<br/>　　一个人睡觉空朗朗　　<br/>　　破门破窗破水缸　　<br/>　　风吹进来我一身光　　<br/>　　孤雁落在沙滩上卧　　<br/>　　难比我妹子心难过　　<br/>　　男瞎子唱：　　<br/>　　日头落在西山沟　　<br/>　　没老婆的哥哥谁收留　　<br/>　　一只风箱空又空　　<br/>　　没老婆的哥哥谁心疼　　<br/>　　上半坡，走半道　　<br/>　　光棍汉受罪谁知道　　<br/>　　毛草房上十八根椽　　<br/>　　谁知道哥哥受艰难　　<br/>　　人家栽葱我栽蒜　　<br/>　　难活不过光棍汉　　<br/>　　一边唱着戏，一边锄着地，就到了夏天，狠狠收了半月麦。天是该雨是雨，该日是日，想不到庚寅年是受活入社的第一年，麦竟丰收得大小田地里的麦穗都差一点压断麦棵的脖。打麦时，满世界都是黄灿灿的麦香味。筹划是打一天麦，分一天粮，不让麦堆在麦场上，可这一分就分了半个月。半个月每家都要往家挑麦子，扛麦子。　　<br/>　　缸满了，囤满了，家里为老人准备了棺材的，就把麦子往那棺材里倒。没有棺材的，就往床上的光席上倒。到末了，再分的小麦没地方倒，各家的墙角和旮旯里都是麦袋儿，连往年盛夏最臭的茅厕里都是了麦香味，最后就把剩下的麦子堆在麦场上的两间麦场屋，就以为入了社，真是过上了天堂日子了。然而，跟着天堂日子来的却是一场大铁灾①。</p></div>
