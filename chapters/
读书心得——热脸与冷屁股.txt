<h1 id="chapter_title_177988"><strong>读书心得——热脸与冷屁股</strong></h1>
<p class="info">
分类：<a class="cate" href="/novel-list-22.html" target="blank">文学作品</a><span class="divide">|</span>作者：<a class="name" href="/pdnovel.php?mod=search&amp;ac=author&amp;srchtxt=李宗吾&amp;searchsubmit=yes"  target="_blank">李宗吾</a><span class="divide">|</span>发布时间：2014-09-24 20:36:41<span class="divide">|</span></p>
<div class="page-body">
<div class="page-content">
<p>我读中学的时候,应该是高二那一年,来了个插班生,后来才知道,他去年就参加高考了,什么都没考上,自己觉得底子潮,干脆就从高二开始插班了.<br/>　　<br/>　　我们当时那个班主任,是个非常拧巴的人,拧巴到几乎全体同学都反对他的程度.所以座位座次基本上是学习好的在前面,在中间,学习不好但听话的可以略略靠前靠中间,学习不好,不听话的,自然靠边站.我是学习还说得过去,但素来不买他的帐的人,自然就到了后面的一个角落里,和一个学习不好的同学同桌,所以现在回想起来,满腹心酸,一点没有同桌的你那样的浪漫记忆来安慰自己受伤的心.<br/>　　<br/>　　<br/>　　<br/>　　前面说的那个插班的同学,因为是个老油子,自然比我们这些人道行深,很快就获得了拧巴老师的欢喜,座位自然随着受喜欢的程度一再向前靠拢,但他自己不喜欢去前面,因为在老师眼皮底下,实在不能有所作为,所以他还是主动要求留在后面.这样我就和他有了接触.<br/>　　<br/>　　后来,这个人考上了一所中专,但他在学校里认了个掌握分配大权的老师做干亲,去了个好单位,然后就一路顺顺当当地做了个小官僚,据说手里有点小权力.<br/>　　<br/>　　于是有同学想去求他办事,但都打了退堂鼓,即使去的,也没有办成什么.为什么呢?因为这个人有句名言横在前面:<br/>　　<br/>　　你觉得和我不错,我觉得和你没嘛.<br/>　　<br/>　　这句话忒狠,换个说法就是,你别拿你的热脸来贴我的冷屁股.<br/>　　<br/>　　<br/>　　<br/>　　我牢记着他这句名言,但现在连他的名字都忘了.我写这点文字的时候,使劲去想他的名字,到底还是没想起来.<br/>　　<br/>　　当然,我是不会拿自己的脸去贴他的屁股的.倒是当年那个学习特差的同桌,现在做了检察官,还偶尔有联系.<br/>　　<br/>　　<br/>　　<br/>　　后来我在社会上混,始终坚持的一个原则就是不去贴人家的屁股.但我发现,这个社会,厚黑学盛行,热脸贴冷屁股的事情太多了.<br/>　　<br/>　　<br/>　　<br/>　　有些事情,你拿自己的一己之愿,想当然地以为可以讨人家欢喜,结果呢,人家以冷屁股对之,唉!<br/>　　<br/>　　世事如此,好多别的事情呢?</p></div>
