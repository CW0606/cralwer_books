<h1 id="chapter_title_177988"><strong>第三十三回　丽句清词三分宿慧　花香灯影一片艳情</strong></h1>
<p class="info">
分类：<a class="cate" href="/novel-list-18.html" target="blank">古典小说</a><span class="divide">|</span>作者：<a class="name" href="/pdnovel.php?mod=search&amp;ac=author&amp;srchtxt=魏秀仁&amp;searchsubmit=yes"  target="_blank">魏秀仁</a><span class="divide">|</span>发布时间：2015-01-01 22:46:37<span class="divide">|</span></p>
<div class="page-body">
<div class="page-content">
<p>　　话说痴珠自入正后，深居西院，或听秋痕弹琴，或瞧秋痕作画，就县前街也少得去了。　<br/>　　这日上元，子秀、子善久不见面，便两人一车，到了秋心院。值门开着，下车走入。见静悄悄的，没个人影；再看月亮门，落把大锁。两人愕然。后来李裁缝出来说起，才知道初二后，秋痕通没回来。两人出来上车。便吩咐赶向秋华堂来。　<br/>　　看门见是熟客，就不通报。两人沿西廊步入月亮门，见厨房里一个打杂，在那里打盹，便悄悄的向西屋窗下走来。正待转人楼下甬道，听得痴珠朗吟道： <br/>　<br/>　　“浮萍大海终飘泊，羞向红颜说报恩。”两人站着脚，又听得秋痕道：“你也有些年纪了，积些余囊，作个买山归隐之计，也是着实打算。再者，你的性情不能随俗，万分做不过荷生，让他得意吧。”痴珠叹一口气道：“我为着家有老母，不得已奔走四方，谋些衣食；不然，我就做和尚。”秋痕道：“你好好做诗，都是我说着闲话，又引起你的心绪来了。”痴珠道：“我这上半四首，已是不及他的原作，再做下去，也没有好句出来，不如算了，不作吧。”秋痕道：“你昨晚说的‘绣榻眠云扶不起，绮窗初日会难逢。三生风絮年来绾，一室天花夜不寒’，都是佳句，怎的不好？”　<br/>　　两人听了半天，正待移步，不想工环从而道出来看见，便报道：“留大老爷和晏太爷来了！”　<br/>　　痴珠迎出，延人客厅。秋痕掀开香色布棉帘招呼。两人觉屋里一阵兰花香扑鼻，就行步入。见窗下四盆素心兰，开有二十余箭，便向书案走来。　<br/>　　案上一幅长笺，狂草一半；子善看了兰花，因取来瞧，上写“奉和本事诗三叠前韵。”子秀念道：　<br/>　　“第一洞天访碧霞，云翘有约总非赊。 <br/>　　鸾笙吹出香窠暖，凤简题成锦字斜。 <br/>　　楚岫朝云开远黛，天台暮雨洗浓华。 <br/>　　寻常小谪人间去，也作秋风得意花。　<br/>　　福慧修来费几生？珊珊仙骨照人清。 <br/>　　衫裁燕尾成双影，扇写蝇头忆定情。 <br/>　　锦瑟相思频入咏，枕屏两地暗呼名。 <br/>　　琼霄指日翔鸾风，别鹤何须带怨声！　<br/>　　番风轮指数迟迟，贮月楼成燕不知。 <br/>　　才子巾箱金粉艳，美人妆盥芷兰思。 <br/>　　娇呼小字猜莲子，爱唱新词谱《竹枝》。 <br/>　　陌上花开归缓缓，荆钗珈服两相宜。　<br/>　　溷我卑栖水外村，天涯回首旧琴樽。 <br/>　　西风铁笛黄泥坂，夜月银筝白下门。 <br/>　　烟柳灞桥留别梦，胭脂北地染新痕。 <br/>　　浮萍大海终飘泊，羞向红颜说报恩！　<br/>　　蓬山风引叹无缘，辜负笺天四十年。 <br/>　　四扇画梅成小影，绣裙簇蝶记游仙。”　<br/>　　子善道：“清艳得很。”子秀笑道：“我们今天做个催租客，打断人家诗兴了。”秋痕道：“他正不高兴，恰好你来，和他谈谈吧。”林喜端上茶来，玉环装着水烟，四人各说了近事。　<br/>　　子秀见上首挂着荷生集《座位》写的一付联对，是：　<br/>　　座列名香，文如满月； <br/>　　家承清德，室有藏书。　<br/>　　中间是心印的一幅画梅横披，横技下贴两纸色笺。便走近一瞧，见是七绝四首，款书“女弟子游畹兰呈草”。便向痴珠道：“你那里又收个会做诗的女弟子？”秋痕笑道：“不就是李太太？”子秀道：“不错，他娘家姓游。”子善也走过来看。因念道：　<br/>　　“华灯九陌照玲珑，掩映朝暾一色红。 <br/>　　最是太平真气象，万人如海日当中。　<br/>　　雕轮宝马度纷纷，百和衣香昨夜薰。 <br/>　　绣帏珠帘都不下，轻尘一任上乌云。　<br/>　　场萧吹暖遍长街，可有游人拾堕钗， <br/>　　满地香尘轻试步，几回珍重踏青鞋。　<br/>　　小幅泥金写吉祥，十枝绎蜡照华堂。 <br/>　　并门多少娇儿女，但愿家家福命长。”　<br/>　　念毕，说道：“李太太也会做诗么？”子善道：“几见诗人的弟子不会做诗？”就掀着卧室帘子，见窗下两盆水仙花，也自盛开；壁上新挂一付联，一幅山水的横披，横技下也粘一色笺。便踱进去．瞧着联一边款书“痴珠孝廉正腕”，一边书“雁门杜梦仙学书”，句是：　<br/>　　诵十万言，有诗书气； <br/>　　翔九千仞，作逍遥游。　<br/>　　当下子秀和痴珠都跟进来。子善道：“采秋竟会写起大字，且有笔力，真是夙慧。”子秀道：“不要说采秋，就秋痕不是大有慧根，怎么几个月工夫，就会做诗呢？”痴珠道：“大约琴棋书画，诗酒文词，都要有点夙根，才能学得来。你看采秋这幅画，不更好么？”子善、子秀瞧着那幅画，是幅工画山水，笔意却极洒落，小楷款书“奉夫子命，为痴珠孝廉作，韩宅侍儿梦仙写”。子善道：“这落款就也新鲜。”旁有小楷一诗，是荷生题的，子秀念道：　<br/>　　“拔地奇峰无限好，在山泉水本来清。 <br/>　　飘然曳杖绝尘事，独向翠微深处行。”　<br/>　　两人再看色笺的诗，上书《水仙花》三字，下书“侍儿刘梧仙呈草”。子善念道：　<br/>　　“云停月落座留香，一缕冰魂返大荒。 <br/>　　银烛高烧呼欲出，仙乎宛在水中央。 <br/>　　好伴吟边与酒边，蓬莱春在画堂前。 <br/>　　烟波倘许侬偕隐，自抱云和理七弦。”　<br/>　　子秀道：“大有寄托。”又看了痴珠的帐缘，是秋痕画的菊，就说道：“秋痕的画菊，竟一天苍老一天了。”　<br/>　　当下秃头回道：“池师爷请爷说话。”痴珠出外间去了。子善随手将案上一个书夹一检，见断笺上有诗两首，瞧是：　<br/>　　对卿乡更觉温柔，雨滞云痴不自由。 <br/>　　胸却比酥肤比雪，可堪新剥此鸡头。　<br/>　　秋波脉脉两无言，擅口香含一缕温。 <br/>　　锦帐四垂银烛背，枕边钦坠个中魂。　<br/>　　又一素纸，上书《题画》，云：　<br/>　　绣帏怎不卸银钩，微识双双艳语柔。 <br/>　　仿佛钗声抛纸上，销魂岂独是天游？　<br/>　　无言只是转星眸，个里情怀不自由。 <br/>　　水溢银河云尚殢，子夫散发最风流。　<br/>　　春雨梨花醉玉楼，双双弹罢卧箜篌。 <br/>　　谁将镜殿铜屏影，付与春风笔底收？　<br/>　　两人一笑。又检得字条，楷书写的是“灯下红儿，真堪销恨。花前碧玉，颇可忘忧”十六字。又色笺两纸，写的是：　<br/>　　埋骨成灰恨未休，天河迢递笑牵牛。 <br/>　　斑雕只系垂杨岸，万里谁能访十洲？　<br/>　　欲人卢家白玉堂，何曾自敢占流光？ <br/>　　可怜夜半虚前席，万里西风夜正长。　<br/>　　龙护瑶窗凤掩扉，含烟惹雾每依依。 <br/>　　何当共剪西窗烛，日暮归来雨满衣。　<br/>　　云鬓无端怨别离，流莺漂荡复参差。 <br/>　　东来西去人情薄，莫枉长条赠所思。　<br/>　　末书：“日来读玉溪生诗，因集得诗如右，呈政吟坛。此中情事，有君有我，有是有非，知足下必能参之也。并希示复，或赐和为望。荷生漫作。”　<br/>　　两人不大解得就中谜语，就检别的来瞧，内还有秋痕的词并手札。词云：　<br/>　　花笺唱酬，曳断情丝千万缕。独对柳梢新月影，算今宵人约黄昏后。眉双绉，奈东君一刹，去矣难留。帘幕锁人愁。风风雨雨，肠断晚妆楼。　<br/>　　又一词云：　<br/>　　花怜小劫，人怜薄命，一样销魂处。香销被冷，灯深漏静，想着闲言语。　<br/>　　两人只看到这一纸，瞥见秋痕掀帘进来，将书夹一抢，说道：“半天没有声息，却原来偷瞧人家机密的书札！”子秀笑道：“事无不可对人言。”子善笑道：“‘人约黄昏后’，怎的可对人言？”就出去了。　<br/>　　到了客厅，雨农要走，痴珠因留三人小饮，并请了萧赞甫。到得黄昏，大家都要出去逛灯，痴珠就不十分强留。　<br/>　　此时里外都点上灯。客厅中点的是两对西番莲洋琉璃灯，里屋两间通点一对湘竹素纱、一边字一边画的灯，正檐下一字儿四对明角灯。一会，月也上来，客厅中两盆碧桃花开得艳艳，映着灯光，就像嫣然欲笑一般。　<br/>　　秋痕将屋里两重棉帘尽行掀起，引着兰花水仙的香。痴珠就领秋痕到秋华堂玩赏一回月，忽然对秋痕道：“你看如此月色，天又不冷，我们何不同到芙蓉洲水阁走一走？”秋痕道：“怕碰着人，不好意思。”痴珠道：“这时候，还有什么人，跑来这冷静地方？”便唤秃头、穆升，先去通知看守的人，教他预备茶水伺候去了。正是：　<br/>　　灯下红儿，花前碧玉。 <br/>　　销恨忘忧，同心一曲。　<br/>　　欲知后事如何，且听下回分解。　</p></div>
