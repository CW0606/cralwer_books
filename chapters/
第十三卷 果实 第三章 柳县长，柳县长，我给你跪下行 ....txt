<h1 id="chapter_title_177988"><strong>第十三卷 果实 第三章 柳县长，柳县长，我给你跪下行 ...</strong></h1>
<p class="info">
分类：<a class="cate" href="/novel-list-23.html" target="blank">文学小说</a><span class="divide">|</span>作者：<a class="name" href="/pdnovel.php?mod=search&amp;ac=author&amp;srchtxt=阎连科&amp;searchsubmit=yes"  target="_blank">阎连科</a><span class="divide">|</span>发布时间：2015-05-23 16:11:24<span class="divide">|</span></p>
<div class="page-body">
<div class="page-content">
<p>“对不起呀柳县长，我对不起你呀柳县长！”　　<br/>　　“奶奶的，一刀砍了你，我一枪崩了你！崩你砍你都不解我的恨。”　　<br/>　　“柳县长，柳县长，我真的对不起你呀柳县长。”　　<br/>　　“跪下来，你们都给我跪下来！”　　<br/>　　“不怪他，不怪石秘书，啥都怪我呢！”　　<br/>　　“滚！你个骚娘们，你这头母猪、母狗、黄鼠狼！”　　<br/>　　“柳县长，别打她你打我好不好？你看她满脸是血了，再打就要打出人命啦。千刀万剐都是我的错，都是我这石秘书的错。”　　<br/>　　“不打她，叫我打你是不是……你以为老子我会放了你……”　　<br/>　　“啊……啊……啊呀……”　　<br/>　　“我撤了你的职……我让你住完监狱再回家里去种地。”　　<br/>　　“你打吧，柳县长，你把我踢死、跺死、踩成肉酱都行哩。”　　<br/>　　“我日你祖宗八辈哩，我现在就让公安局把你送进监狱里。我一句话就能让你家破人亡，就能让你们名誉扫地，成过街老鼠，在双槐让你们寸步难行；在双槐让你们逃荒要饭都没地处儿去。”　　<br/>　　“求求你，别打他了，你看他都昏了过去啦；老柳呀，柳县长，求你还来打我行不行。”　　<br/>　　……　　<br/>　　“日你祖奶奶，你给我说句实话吧……现在你出门，人人都说你是县长的夫人哩，都称你是夫人你知道不知道？”　　<br/>　　“知道哩，可我不想当夫人。我就想当个一般人的媳妇哩，下班了烧烧饭，拖拖地；男人坐在沙发上看报纸，我在灶房里忙个不停儿。待饭菜端到桌上了，他放下报纸和我一块去吃饭。吃过了，我坐在沙发上看报纸，他去灶房洗锅洗碗了。洗完了，两个人一道坐在沙发上看电视，说说话，末了就都上床去睡了。”　　<br/>　　“柳县长，你就成全了我们吧。你不成全我们，我俩就跪到天亮不起来。”　　<br/>　　“水哩？水哩？祖奶奶，这家里连一口水喝都没有。”　　<br/>　　“没水了……我这就给你烧，这就给你倒。”　　<br/>　　“我日你祖奶奶，没想到提携你当我的秘书，你反倒伤透了我的心，买不回列宁遗体都没有你给我的打击大。”　　<br/>　　“对不起你了柳县长，真的是对不起你呀柳县长。”　　<br/>　　“行啦、行啦，你把额门磕出血我柳县长都不会原谅你。”　　<br/>　　“我不求你原谅呢，我罪有应得哩。”　　<br/>　　“喝水吧……有些热……你先凉一凉。”　　<br/>　　“茶叶呢？”　　<br/>　　“泡绿茶还是红茶呀？”　　<br/>　　“随你妈的便。”　　<br/>　　“那就绿茶吧，绿茶消火呢。”　　<br/>　　“站起来，你说咋办吧。”　　<br/>　　“柳县长，你不说句原谅的话，我死也不会站起来。”　　<br/>　　“那就跪着吧，说你想咋办吧。”　　<br/>　　“求柳县长成全了我们俩……”　　<br/>　　“成全了我们吧，不成全我和他一块跪死在你面前。”　　<br/>　　“说说咋个成全法。”　　<br/>　　“让我俩结婚吧，要在双槐丢了你的脸，你把我俩的工作调到天南地北都行哩。”　　<br/>　　“柳县长，我们不会忘了你的大恩大德哩。我跟着你当了这么多年的秘书啦，没有人比我更知道你心里想要啥儿哩。你成全了我们俩，我会让全县城的人都给你跪下磕头呢。我知道列宁遗体买不回来啦，买不回来我也会让全县城的人给你跪下磕头哩，让全县人民见了你就给你跪下磕头哩。不信了你试试，我明儿天就让大街上无论谁见你都跪下给你磕个头。让新城、老城的人都在正堂屋里挂着你的画像行不行。”　　<br/>　　“哼……你以为你是神仙是不是？对你说，老天爷都没这个能耐啦。”　　<br/>　　“柳县长，我说到做到哩。”　　<br/>　　“滚！你们两个都给我滚，滚得越远越好哩。”　　<br/>　　……　　<br/>　　“你半年没有回家了，我想……再陪你一夜说说话。”　　<br/>　　“不用说，这家里的东西想要啥了你全都拿去吧。”　　<br/>　　“我啥都不要哩，我只把我爹的像带走就行啦。”　　<br/>　　“带走吧，想要啥你就带走吧。”　　<br/>　　“那我们就走啦。”　　<br/>　　“走吧，快点走掉吧。再别让我看见你们俩。”　　<br/>　　“谢你了，柳县长……我知恩必报哩，我会记着你的大恩大德呢。我明儿天就让全县人民跪下给你磕头哩，让各家各户把你当成神敬哩。”</p></div>
