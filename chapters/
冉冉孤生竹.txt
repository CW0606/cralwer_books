<h1 id="chapter_title_177988"><strong>冉冉孤生竹</strong></h1>
<p class="info">
分类：<a class="cate" href="/novel-list-19.html" target="blank">古代诗词</a><span class="divide">|</span>作者：<a class="name" href="/pdnovel.php?mod=search&amp;ac=author&amp;srchtxt=佚名&amp;searchsubmit=yes"  target="_blank">佚名</a><span class="divide">|</span>发布时间：2014-09-02 21:18:46<span class="divide">|</span></p>
<div class="page-body">
<div class="page-content">
<p>【原文】<br/>　　冉冉孤生竹⑴，结根泰山阿⑵。<br/><br/>　　与君为新婚⑶，兔丝附女萝⑷。<br/><br/>　　兔丝生有时，夫妇会有宜⑸。<br/><br/>　　千里结远婚，悠悠隔山陂⑹。<br/><br/>　　思君令人老，轩车来何迟⑺！<br/><br/>　　伤彼蕙兰花⑻，含英扬光辉⑼；<br/><br/>　　过时而不采，将随秋草萎⑽。<br/><br/>　　君亮执高节⑾，贱妾亦何为⑿？<br/><br/>【注释】<br/>　　⑴冉冉：柔弱貌。　<br/>　　⑵泰山：即“太山”，犹言“大山”，“高山”。阿：山坳。这两句是说，柔弱的孤竹生长在荒僻的山坳里，借喻女子的孤独无依。　<br/>　　⑶为新婚：指订婚，非出嫁。　<br/>　　⑷菟丝：一种旋花科的蔓生植物，女子自比。女萝：一说即“松萝”，一种缘松而生的蔓生植物；以比女子的丈夫。这句是说二人都是弱者。　<br/>　　⑸宜：适当的时间。这两句是说，菟丝及时而生，夫妇亦当及时相会。　<br/>　　⑹悠悠：遥远貌。山陂：山坡。这二句是说路途遥远，结婚不易。　<br/>　　⑺轩车：有篷的车。这里指迎娶的车。这二句是说，路远婚迟，使她容颜憔悴。　<br/>　　⑻蕙、兰：两种同类香草。女子自比。　<br/>　　⑼含英扬光辉：花含苞待放。英，犹“花”。　<br/>　　⑽萎：枯萎，凋谢。这四句是说，蕙兰过时不采，它将随着秋草一同枯萎了。这是对婚迟的怨语。　<br/>　　⑾亮：同“谅”，料想。　<br/>　　⑿贱妾：女子谦称。这两句是说，君想必守志不渝，我又何苦自艾自怨。这是自慰之词。<br/><br/>【鉴赏】<br/>　　此诗或云是婚后夫有远行，妻子怨别之作。然细玩诗意，恐不然。或许是写一对男女已有成约而尚未成婚，男方迟迟不来迎娶，女方遂有种种疑虑哀伤，作出这首感情细腻曲折之诗。<br/>　　“冉冉孤生竹，结根泰山阿。”竹而曰“孤生”以喻其孑孑孤立而无依靠，“冉冉”是柔弱下垂的样子。这是女子的自喻。“泰山”即“太山”，大山之意。“阿”是山坳。山是大山，又在山阿之处，可以避风，这是以山比喻男方。《文选》李善注曰：“结根于山阿，喻妇人托身于君子也。”诚是。<br/>　　“与君为新婚，兔丝附女萝。”兔丝和女萝是两种蔓生植物，其茎蔓互相牵缠，比喻两个生命的结合。《文选》五臣注：“兔丝女萝并草，有蔓而密，言结婚情如此。”从下文看来，兔丝是女子的自喻，女萝是比喻男方。“为新婚”不一定是已经结了婚，正如清方廷珪《文选集成》所说，此是“媒妁成言之始”而“非嫁时”。“为新婚”是指已经订了婚，但还没有迎娶。<br/>　　“兔丝生有时，夫妇会有宜。”这还是以“兔丝”自喻，既然兔丝之生有一定的时间，则夫妇之会亦当及时。言外之意是说不要错过了自己的青春时光。<br/>　　“千里远结婚，悠悠隔山陂。”从这两句看来，男方所在甚远，他们的结婚或非易事。这女子曾企盼着，不知何时他的车子才能到来，所以接下来说：“思君令人老，轩车来何迟！”这首诗开头的六句都是比，这四句改用赋，意尽旨远，比以上六句更见性情。<br/>　　“伤彼蕙兰花，含英扬光辉。过时而不采，将随秋草萎。”这四句又用比。蕙和兰是两种香草，用以自比。“含英”是说花朵初开而未尽发。“扬光辉”形容其容光焕发。如要采花当趁此时，过时不采，蕙兰亦将随秋草而凋萎了。这是希望男方趁早来迎娶，不要错过了时光。唐杜秋娘《金缕衣》：“花开堪折直须折，莫待无花空折枝。”与此两句意思相近。<br/>　　最后二句“君亮执高节，贱妾亦何为？”张玉谷说：“代揣彼心，自安己分。”诚然。这女子的疑虑已抒写毕尽，最后遂改为自我安慰。她相信男方谅必坚持高尚的节操，一定会来的，那么自己则不必怨伤。 </p></div>
