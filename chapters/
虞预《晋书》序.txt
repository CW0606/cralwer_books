<h1 id="chapter_title_177988"><strong>虞预《晋书》序</strong></h1>
<p class="info">
分类：<a class="cate" href="/novel-list-22.html" target="blank">文学作品</a><span class="divide">|</span>作者：<a class="name" href="/pdnovel.php?mod=search&amp;ac=author&amp;srchtxt=鲁迅&amp;searchsubmit=yes"  target="_blank">鲁迅</a><span class="divide">|</span>发布时间：2014-10-06 21:25:01<span class="divide">|</span></p>
<div class="page-body">
<div class="page-content">
<p>《隋志》：《晋书》二十六卷，本四十四卷，讫明帝（2），今残缺，晋散骑常侍虞预撰。《唐志》：五十八卷。《晋书》《虞预传》：著《晋书》四十余卷。与《隋志》合，《唐志》溢出十余卷，疑有误。本传又云：预字叔宁；征士喜（3）之弟也。本名茂，犯明穆皇后讳（4），改。初为县功曹，见斥。太守庾琛（6）： <br/>　　命为主簿。纪瞻（6），复为主簿，转功曹史。察孝廉，不行。 <br/>　　安东从事中郎诸葛恢（7），参军庾亮（8）等荐预，召为丞相行参军兼记室。遭母忧，服竟，除佐著作郎。大兴中，转琅邪国（9）常侍，迁秘书丞，著作郎。咸和中，从平王含（10），赐爵西乡侯。 <br/>　　假归，太守王舒（11）请为谘议参军。苏峻（12）平，进封平康县侯，迁散骑侍郎，著作如故。除散骑常侍，仍领著作。以年老归，卒于家。 <br/><br/>==========================================================================<br/><br/>　　 　本篇据手稿编入，原无标点。当写于一九一三年三月。 <br/>　　虞预，晋代余姚（今属浙江）人。所著《晋书》四十四卷，已佚； <br/>　　又著有《会稽典录》二十篇，《诸虞传》十二篇，并佚。鲁迅所辑虞氏《晋书》一卷，未印行。 <br/>　　（2）　明帝　东晋明帝司马绍（299—325），三二二年至三二五年在位。 <br/>　　（3）　征士喜　指虞喜（281—356），字仲宁，晋代学者。朝廷三次征拜博士等官，俱不就。著有《安天论》、《志林新书》等。 <br/>　　（4）　明穆皇后　指晋明帝后庾文君。按文中说虞预本名犯明穆皇后讳，《晋书·虞预传》作“犯明穆皇后母讳”。 <br/>　　（5）　庾琛　字子美，颍川鄢陵（今属河南）人，明穆皇后父。西晋末年任会稽太守，官至丞相军谘祭酒。 <br/>　　（6）　纪瞻（253—324）　字思远，丹阳秣陵（今江苏南京）人。 <br/>　　西晋末年任会稽内史，官至骠骑将军。 <br/>　　（7）　诸葛恢（265—326）　字道明，琅玡阳都（今山东沂南）人。 <br/>　　曾任安东将军司马睿（即后来的晋元帝）属下的从事中郎，后官至尚书右仆射。 <br/>　　（8）　庾亮于西晋愍帝建兴（313—316）年间任丞相司马睿的参军。 <br/>　　（9）　琅邪国　琅邪亦作琅琊。西晋时，琅邪王封地在今山东临沂地区；东晋时，侨置于今江苏句容地区。太兴二年（319）虞预任琅邪国常侍，当时琅邪王为元帝子司马裒。 <br/>　　（10）　王含　字处弘，临沂（今属山东）人，东晋大将军王敦之兄。官至骠骑大将军，随王敦叛乱，失败被沉水死。按明帝太宁二年（324）平王含，在成帝咸和（326—334）前。 <br/>　　（11）　王舒　字处明，临沂人。东晋太宁末、咸和初任抚军将军、会稽内史。因平苏峻有功，进封彭泽县侯。 <br/>　　（12）　苏峻　字子高，掖（今山东掖县）人。东晋元帝时官至冠军将军。咸和二年（327）起兵叛乱，次年兵败被杀。 </p></div>
