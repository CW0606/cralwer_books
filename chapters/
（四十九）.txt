<h1 id="chapter_title_177988"><strong>（四十九）</strong></h1>
<p class="info">
分类：<a class="cate" href="/novel-list-23.html" target="blank">文学小说</a><span class="divide">|</span>作者：<a class="name" href="/pdnovel.php?mod=search&amp;ac=author&amp;srchtxt=魏巍&amp;searchsubmit=yes"  target="_blank">魏巍</a><span class="divide">|</span>发布时间：2015-04-23 23:35:11<span class="divide">|</span></p>
<div class="page-body">
<div class="page-content">
<p>毛泽东是第二天赶到安顺场的。 <br/>　　这天又是一个天色阴沉的日子，大渡河的上空堆满了浓重的灰云。他站在马鞍山上观察了许久，才走下山来。周恩来、朱德等人，已经到先遣队的司令部去了。 <br/>　　“我倒想先看看这条怪河！” <br/>　　毛泽东说着，就先同警卫员小沈来到渡口。 <br/>　　他自然见过不少浩瀚的江水，但往大渡河边一站，那磅礴的声势，仍然使他惊叹。小沈顺手拣起一根柴棍，往里一丢，那根柴棍瞬息间飞射出四五米远。小沈又扔下一片树叶，叶子随着车轮般的大漩涡嘀溜乱转，不一时就沉入河底去了。可是，毛泽东望望那只载运红军战士的木船，它却在惊涛骇浪中浮浮沉沉，走得相当迟慢。 <br/>　　河岸上有十几个船工，正坐在草地上休息。他们大都穿着短裤，露着赤铜色的肌肉，在那里抽烟说笑。 <br/>　　毛泽东走过去，往他们中间随便一坐，笑着说： <br/>　　“老乡，你们都是摇船的吧！” <br/>　　大家都笑着点头。毛泽东说： <br/>　　“多亏你们帮忙呵！不然可真是过不去咧。” <br/>　　说着，掏出纸烟来给了每人一支，自己也点着抽起来。 <br/>　　昨天刘伯承见到的那个满脸胡子的壮汉也在其中。他呵呵笑着说： <br/>　　“穷人的队伍嘛，我们帮一点小忙还不应该？” <br/>　　他收起小旱烟管，把毛泽东给他的纸烟点着，抽了一口： <br/>　　“你们红军可真是不错！昨天开第一船，敌人一响枪，你们的战士就说，老板，你们往后一点，让我们拉船；到了船上，子弹乱飞，我害怕了，他们又说，老板不要怕，打不到你，说着就站起来挡着我们。真是好样儿的！” <br/>　　毛泽东快慰地笑了笑，远远望见那只船已经到了对岸，七八个船工又开始艰难地向上拉着。 <br/>　　“过一船得多长时间？”他问。 <br/>　　“一来一往总得个把钟头。”人们纷纷回答。 <br/>　　“为么子那么慢哪？” <br/>　　“因为水太急呀！”那个壮汉往上游一指，“你看，未曾开船，我们得先把船往上拉到周家碾房，这样斜着过去，才能开到对岸渡口。到了那里，还得往上拉半里路，才能开到这个渡口。” <br/>　　毛泽东的眉头一皱，沉思了一阵，没有说话。他站起来，回转身指着安顺场后面的那座圆包包山，问： <br/>　　“那就是营盘山吗？” <br/>　　“对，对，那就是营盘山。” <br/>　　这座山既不奇特，也不十分高大，几乎是个平顶。令人惊异的是，山坡上荒坟垒垒，几乎满眼都是。毛泽东问： <br/>　　“这都是什么人的坟哪，这么多！” <br/>　　“这就是太平军的坟嘛！”人们纷纷回答。 <br/>　　“我从马鞍山下来，一路上看到很多坟，也是太平军的吗？” <br/>　　“是的，他们在这里死了一万多人呢！” <br/>　　那个壮汉插进来说： <br/>　　“你到洗马姑、大树堡看看，那里坟也很多，光大树堡就杀了两千人呢！到现在，夜深了，还听见他们哭哩！” <br/>　　“不会吧！” <br/>　　“真的，特别是刮风下雨的夜里，他们一边哭，一边还叫： <br/>　　‘报仇呵！报仇呵！’我们都听到过的！” <br/>　　毛泽东垂下头来，没有讲话。 <br/>　　这时，刘伯承从安顺场街上走过来，到了毛泽东身边打了一个敬礼，笑着问： <br/>　　“毛主席，你怎么跑到这里来了？大家都在那边等你呢！” <br/>　　毛泽东笑着说： <br/>　　“大家都说这条河很凶，我也想看看。” <br/>　　说过，又亲热地望着刘伯承，说： <br/>　　“伯承，金沙江的船叫你夺过来了，大渡河的船又叫你夺过来了，你是用的什么鬼办法呀！” <br/>　　刘伯承听出来毛泽东是在表扬他，那只独眼在眼镜后面眨了眨，有点不好意思地笑着说： <br/>　　“这简直是鬼使神差，是敌人一个营长把船送过来的。” <br/>　　接着，他把赖执中的故事说了一遍，引得毛泽东哈哈大笑。 <br/>　　刘伯承指着对岸，把昨天的战斗情况汇报了一遍，尤其把十七勇士和赵章成、土佬的事迹讲得绘声绘色。毛泽东听了，神采飞扬，不绝地赞叹。 <br/>　　接着，他们离开河岸，向安顺场街上缓步走去。 <br/>　　“现在，渡过多少人了？”毛泽东边走边问。 <br/>　　“船太小，每只只能渡四十多人。”刘伯承叹了口气，“现在一团刚刚渡完。” <br/>　　“还能找到船吗？” <br/>　　“听说下游有两条船，也小得很。” <br/>　　“是呵，再有两条也不行呵。”毛泽东扳起指头说，“金沙江是六只木船，比这个船大，还渡了九天九夜。照这样子，恐怕要渡一个月吧！我给你说，伯承，薛岳的五十三师前三四天就从会理追上来了，离这里也不过几天路程。” <br/>　　“是的，刚才总司令、恩来同志也都觉得太迟慢了。” <br/>　　毛泽东脸色严肃，缓缓地说： <br/>　　“这就是说，我们并没有摆脱石达开那样的险境！” <br/>　　“是的。”刘伯承严肃地点了点头。 <br/>　　毛泽东仰起脸望望天空和山梁上的黑云： <br/>　　“看起来，天恐怕还要落雨。……浮桥完全不能架吗？”“不行。”刘伯承摇摇头，“我们试过了，木排刚刚放下就冲跑了。” <br/>　　“除了泸定桥，还有别的桥吗？” <br/>　　“没有。” <br/>　　毛泽东沉思良久，决断地说： <br/>　　“看来，我们非夺取泸定桥不可！” <br/>　　“听说那桥很特殊，只那么几根悬空的铁索，架着一些板子。离这里还有整整三百二十里路。” <br/>　　“那也要夺取！还必须要快！”毛泽东语气坚决，“因为我估计，泸定桥方面敌人也要增兵。” <br/>　　“是的。” <br/>　　“我看最好的办法是兵分两路，夹江而上。这样，敌人就不好守了。” <br/>　　刘伯承的眼里闪出光彩，连声说：“是的，是的。” <br/>　　在安顺场街外，毛泽东放慢了脚步，靠近刘伯承说： <br/>　　“伯承，我给你说，这些天，我的心一直悬着，就是现在也没有放下。我曾作过最坏的打算，即使过不了大渡河，我们就绕到西康去，也决不会学石达开的。” <br/>　　刘伯承望着毛泽东的眼睛，觉得那里面熠熠闪光，闪射着一股极其倔犟的蛮劲，一种不可战胜的光辉。 <br/>　　前面已是设在一家店铺的先遣队司令部，很远就听见里面传出朱德、周恩来等人朗朗的笑声。</p></div>
