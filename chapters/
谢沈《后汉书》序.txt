<h1 id="chapter_title_177988"><strong>谢沈《后汉书》序</strong></h1>
<p class="info">
分类：<a class="cate" href="/novel-list-22.html" target="blank">文学作品</a><span class="divide">|</span>作者：<a class="name" href="/pdnovel.php?mod=search&amp;ac=author&amp;srchtxt=鲁迅&amp;searchsubmit=yes"  target="_blank">鲁迅</a><span class="divide">|</span>发布时间：2014-10-06 21:24:48<span class="divide">|</span></p>
<div class="page-body">
<div class="page-content">
<p>《隋志》：《后汉书》八十五卷，本一百二十二卷，晋祠部郎谢沈撰。《唐志》：一百二卷，又《汉书外传》十卷（2）。《晋书》《谢沈传》（3）：沈字行思，会稽山阴人。郡命为主簿，功曹，察孝廉（4），太尉郄鉴（5）辟，并不就。会稽内史何充（6）引为参军，以母老去职。平西将军庾亮（7）命为功曹，征北将军蔡谟（8）牒为参军，皆不就。康帝（9）即位，以太学博士征，以母忧去职。服阕，除尚书度支郎。何充庾冰（10）并称沈有史才，迁著作郎，撰《晋书》三十余卷。会卒，年五十二。沈先著《后汉书》百卷及《毛诗》（11）《汉书外传》，所著述及诗赋文论皆行于世，其才学在虞预（12）之右。案《隋志》无《外传》者，或疑本在《后汉书》百二十二卷中，《唐志》乃复析出之，然据本传当为别书，今无遗文，不复可考；惟《后汉书》尚存十余条，辄缀辑为一卷。 <br/><br/>==========================================================================<br/><br/>　　 　本篇据手稿编入，原无标点。当写于一九一三年三月。 <br/>　　谢沈（292—344）《后汉书》，清代姚之骃《〈后汉书〉补逸》和汪文台《七家〈后汉书〉》中各有辑本一卷。鲁迅辑本未印行。 <br/>　　（2）　《唐志》　这里兼指《旧唐书·经籍志》和《新唐书·艺文志》。《汉书外传》，《旧唐书·经籍志》著录：“《后汉书》，……一百二卷，谢沈撰。《后汉书外传》，十卷，谢沈撰。”《新唐书·艺文志》著录“谢沈《后汉书》，一百二卷，又《外传》十卷。” <br/>　　（3）　《晋书》　纪传体晋代史，唐代房玄龄等著，一三○卷。 <br/>　　《谢沈传》见该书卷八十二。 <br/>　　（4）　孝廉　“孝悌廉洁科”的简称，汉代选拔官吏的科目之一，每年由郡举“孝廉”，合格者即授予官职。 <br/>　　（5）　郄鉴（269—339）　字道徽，高平金乡（今属山东）人，晋成帝咸康四年（338）任太尉。 <br/>　　（6）　何充（292—346）　字次道，庐江灊（今安徽霍山）人。晋成帝时任会稽内史，官至尚书令。 <br/>　　（7）　庾亮（289—340）　字元规，颍川鄢陵（今属河南）人。晋明帝穆皇后之兄，成帝时封平西将军。 <br/>　　（8）　蔡谟（281—356）　字道明，陈留考城（今河南兰考）人，晋成帝咸康五年（339）封征北将军。 <br/>　　（9）　康帝　东晋康帝司马岳（322—344），三四二年至三四四年在位。 <br/>　　（10）　庾冰（296—344）　字季坚，庾亮之弟，官至中书监。 <br/>　　（11）　《毛诗》　西汉毛亨和毛苌所传《诗经》。《隋书·经籍志》载：梁代有谢沈所注《毛诗》二十卷，《毛诗释义》、《毛诗义疏》各十卷。三书皆亡。 <br/>　　（12）　虞预　参看本书《虞预〈晋书〉序》及其注 。 </p></div>
