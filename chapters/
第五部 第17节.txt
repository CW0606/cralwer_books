<h1 id="chapter_title_177988"><strong>第五部 第17节</strong></h1>
<p class="info">
分类：<a class="cate" href="/novel-list-23.html" target="blank">文学小说</a><span class="divide">|</span>作者：<a class="name" href="/pdnovel.php?mod=search&amp;ac=author&amp;srchtxt=周大新&amp;searchsubmit=yes"  target="_blank">周大新</a><span class="divide">|</span>发布时间：2015-04-25 21:58:07<span class="divide">|</span></p>
<div class="page-body">
<div class="page-content">
<p>卓远赶到汇文小学接月儿的时候，太阳早已躲得没了踪影，偌大的已开始有暮色飘动的小学校院里，只有月儿孤零零站在那儿。卓炯已参军随部队南下走了，卓家又只剩下了老两口和月儿三个人。 <br/>　　“爷爷怎么才来？”月儿的小嘴噘着，显然在生气。 <br/>　　卓远笑笑，卓远先把月儿的书包拿过挂在自己的肩上，尔后拉起月儿的手，一边拄了拐杖帮助那只残了的左腿挪动一边回答：“爷爷在和蔡市长商谈。” <br/>　　“啥叫商谈？” <br/>　　“商谈就是为一件事商量交谈，是一种双方平等的谈话。” <br/>　　“为啥事商谈？” <br/>　　“为当官，蔡市长想叫爷爷当官。” <br/>　　“当啥官？” <br/>　　“就是管各种学校的官。那些学校中有像你们汇文这样的小学，也有初中、高中和师范。” <br/>　　“你答应了？” <br/>　　“没有。” <br/>　　“当官不好吗？” <br/>　　“当官有好处也有坏处。好处是有权办自己想办的事情，能谋到个人的名誉和利益；坏处是失去了自由思索的权力。再说，爷爷已经老了，爷爷这辈子从来没到哪届政府里当过官。” <br/>　　“啥叫自由思索？” <br/>　　“就是不受限制的思索，也就是思索时不受框框的限制，就像你晚上睡觉时做梦，怎么梦都行。” <br/>　　“当了官为啥就不能自由思索？” <br/>　　“因为官一般都有上级，而且有纪律，下边的官和上边的官在思索的问题和深度上通常要求一致。” <br/>　　“人为啥要自由思索？” <br/>　　“因为人只有自由思索才能认识自己，认识社会，认识自然，认识宇宙，才能有发现，有发明，有创造，有进步。” <br/>　　“爷爷你为啥要自由思索？” <br/>　　“爷爷是知识者，知识者在社会上就担负着自由思索的任务。这是一种分工，像你姥爷织丝厂里的工人，他们的主要任务是织丝绸，那是社会给他们的分工。爷爷要是不去自由思索问题，爷爷就是失职，就是偷懒。” <br/>　　“你都思索了啥？” <br/>　　“爷爷啥都思索，因为是自由思索，思索的问题可以很多。比如爷爷眼下正思考着怎样办好南阳的教育，办哪些学校好，每一种学校开设哪些课程，南阳急缺的人才是哪些……” <br/>　　“那你思索了这些问题给谁说呢？” <br/>　　“爷爷可以写成文章在报纸上发表；也可以直接给当官的说。” <br/>　　“人家会听？” <br/>　　“会的，因为爷爷是教育委员会的顾问。” <br/>　　“啥叫顾问？” <br/>　　“顾问就是人家有事情弄不明白了要问你，你去给人家解答，就像你做作业时遇到难题来问我和你奶奶，我们给你解答一样。” <br/>　　“顾问到啥时候？” <br/>　　“还不知道，这要看双方的意愿，假若我不想当了或是他们不愿聘我当了，这顾问就干到头了。” <br/>　　“干到头了咋办？” <br/>　　“就在家里住着啊，反正爷爷也老了。” <br/>　　“那不是奶奶么？她站在门口干啥？” <br/>　　“她在等我们，哎哟，总算到家了。” <br/>　　“你哎哟什么？” <br/>　　“我哎哟是因为你把爷爷问得太累了……” <br/>　　第二天是个星期日，天刚开始酝酿着变黑的时辰，一些在空中飞动的鸟儿还可辨清，街边的一些人家才传出响亮的喝红薯稀饭的响声，卓远就一手拄着拐杖一手拉着月儿，开始了例行的傍晚散步。 <br/>　　气力是让岁月一点一点地拿走了，走不多远卓远的喘息就变得十分急促，他不得不时时停下步子，让急喘慢慢平息下去。“爷爷，你为啥总是站住？”月儿对这种走走停停不甚满意。 <br/>　　“爷爷老了，走不动了。” <br/>　　“我还没有长大，你咋就老了呢？” <br/>　　卓远笑了，“是呀，是呀，爷爷不该老。” <br/>　　“等我长大了你再老，那时我好伺候你！” <br/>　　“谁这样孝顺呀？”爷孙俩的谈话突然被人打断，卓远抬头看时，才见是蔡承银迎面走来，于是点头招呼：“忙哩？” <br/>　　“我刚去看了几所已开学的学校，教学工作做得不错，这是你管的事儿，我该向你——” <br/>　　“我爷爷不做官！”月儿大概以为承银是在说做官的事，大胆地截断了他的话。 <br/>　　“不做官做啥？”承银转而笑望着月儿问。 <br/>　　“做一个自由思索者！”月儿挺了挺身子答。 <br/>　　“哦，”承银抬脸看着卓远：“这么说，你爷爷最近一定又思索了什么重要问题。能告诉我吗，卓先生？”承银的脸上已是一副郑重之态。 <br/>　　“当……然。”卓远犹豫了一下，“我最近想的一件事是：你们掌权之后要长期坚持做下去的最重要的事是什么。这可能是多此一举，你们肯定早就明白！” <br/>　　“我很想再听听你的思索结果！” <br/>　　“摆在你们面前的事情肯定很多，但发展经济才是你们时时不能忘掉的最重要的事情。因为只有这件事做好了，百姓才能有吃有住有穿，国力才能强盛，才能不让日本入侵中国这类事情再次发生！我想提醒你的是两点：其一，中国历朝历代的掌权者在上台之初，差不多都明白要把发展经济当作要务，但实际上都是没过多久，就把这事忘了；其二，日本投降后，我们两国的经济可以说是在一个起点上，但现在他们已经低头不声不响地建设了三年——这三年间我们还在打仗。与他们相比，我们已经失去了三年的时间。我们不能再失去了，不然的话，他们就会在国力上再次占尽优势，那样，说不定就会有又一次入侵！” <br/>　　承银的身子一震。 <br/>　　“日本人独特的民族认同感，日本民族社会的强大凝聚力，日本社会唤起共同目的、共同意识的特殊能力，这些都是不可小视的！你们作为执政者，必须有一种紧迫感！日本已经两次打进了中国，弄不好还会有第三次的！而且你要知道，穿军服的军队不是唯一的军队！” <br/>　　“哦？” <br/>　　“根据历史提供的事实，促使掌权者不再把自己关注的中心放在经济发展上的缘由通常有——” <br/>　　“爷爷，我不想听，我要去麻糖店——！”月儿将爷爷的话音打断，扯着爷爷的手要走。 <br/>　　“对不起，就说到这儿。我这可能是杞人忧天，只是供你参考罢了。” <br/>　　卓远拉着月儿走了，走出很远之后，他无意中回头看见，承银还定定地站在原地。莫非他真的把我的话听进去了？他是在想那些话么？我那些话多少有些逆耳，能听进逆耳之言的掌权者才有希望掌牢权力！但愿他能记住那些话，关注经济发展，关心百姓疾苦，从而使国力强盛，人民幸福……</p></div>
