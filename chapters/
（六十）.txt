<h1 id="chapter_title_177988"><strong>（六十）</strong></h1>
<p class="info">
分类：<a class="cate" href="/novel-list-23.html" target="blank">文学小说</a><span class="divide">|</span>作者：<a class="name" href="/pdnovel.php?mod=search&amp;ac=author&amp;srchtxt=魏巍&amp;searchsubmit=yes"  target="_blank">魏巍</a><span class="divide">|</span>发布时间：2015-04-23 23:37:48<span class="divide">|</span></p>
<div class="page-body">
<div class="page-content">
<p>彭德怀也象全军的指战员一样，对当前不战不进的局面闷闷不乐。这天，他正坐在一家藏民的木楼上闷着头考虑什么，忽听一个参谋在电话上报告，说四方面军张国焘的秘书前来探望，便坐在楼上等候着。 <br/>　　不一时，警卫员就将一个人领上楼来。这人向彭德怀恭恭敬敬而又很潇洒地打了一个敬礼，接着说： <br/>　　“我是张主席的秘书黄超，是奉张主席之命来慰问彭军团长的。” <br/>　　彭德怀一打量来人，是个相当年轻漂亮的青年军官。他长着一副曼长脸，面孔白皙，两只闪闪的大眼睛，透露着聪明灵活，善知人意。彭德怀同他握了手，就请他在火塘边坐下。 <br/>　　黄超一坐下，便滔滔不绝，称赞彭德怀是海内名将，无人不晓，自己作为后生小辈已倾慕多年，今日是相见恨晚了。 <br/>　　彭德怀见他说个没完，就说： <br/>　　“都是自己人嘛，不要太客气了。” <br/>　　“这怎么是客气呢！”黄超讲得更加来劲，“一方面军西征行程一万八九千里，彭军团长斩关夺隘，声震遐迩，不要说自己人，就是敌人也闻风丧胆。张主席平日常谈起彭军团长，他觉得这地方生活很苦，所以叫我送点东西来，表示慰劳。” <br/>　　“那我就谢谢他了。”彭德怀说。 <br/>　　黄超转过头看了看警卫员已经出去，就试探着问： <br/>　　“彭军团长，你是不是参加过一个会理会议？”“参加过。”彭德怀答道；一面心中暗想：“他为什么要问这个？” <br/>　　“你那次处境不大好吧？”黄超闪着一双机灵的眼睛。 <br/>　　“处境？什么处境？” <br/>　　彭德怀对这位年轻人提出的问题感到意外。 <br/>　　黄超笑了笑，说： <br/>　　“一个人遭到不白之冤，总是叫人不愉快的。” <br/>　　彭德怀带有几分粗野地望了黄超一眼： <br/>　　“无非是受了一点批评，这在我们党内也很平常。”“批评自然是常事，”黄超笑着说，“如果太不公平，也会叫人沮丧。” <br/>　　“没什么！”彭德怀紧接上去，“仗没有打好，有点右倾情绪，受点批评，这是很自然的。” <br/>　　说到这里，彭德怀盯住黄超： <br/>　　“怎么，你要了解会理会议？中央给你谈了？” <br/>　　黄超涨红着脸说： <br/>　　“不不，我只是随便问问。……张主席是很知道你的，也很关心……” <br/>　　彭德怀木着脸，没有表情，冷倔倔地捅出一句： <br/>　　“我们过去没见过面。” <br/>　　黄超的勇敢进攻受了挫折，伤了几分锐气，为了完成任务，不得不继续鼓劲。他眼珠转了几转，便改了话题。 <br/>　　“一、四方面军会合以后，确实力量大了。但是战略方针还要正确。如果这方面发生偏差，兵力再大也不行。” <br/>　　彭德怀脸上露出一丝轻蔑的笑意： <br/>　　“黄秘书，你看怎么才算正确？” <br/>　　黄超不免有点尴尬，带着几分忸怩地说： <br/>　　“不是我看，是张主席考虑：还是南下才是上策。他曾跟我说，‘欲北伐必先南征’。” <br/>　　“那是什么情况？”彭德怀轻蔑地一笑，“那是诸葛亮巩固蜀国后方的办法。我们现在连根据地都没有，哪里有这样的后方？” <br/>　　黄超挨了一棒，心里已有几分恼怒，但在这个威严人物的面前，毕竟不敢放肆，就客气地反驳道： <br/>　　“彭军团长，北进也不那么容易吧，胡宗南是蒋介石的嫡系，武器装备是最精良的，战斗力很不一般。还有马家军的骑兵，不仅装备好，而且训练有素，每人一把大马刀，在草原上跑起来简直象……” <br/>　　彭德怀脸有愠色，立刻打断他： <br/>　　“你是叫他们吓昏了吧！” <br/>　　黄超满脸通红。沉了一下，继续争辩说： <br/>　　“对形势的看法是需要冷静、客观才能得出正确答案的。张主席多次说，当前苏维埃运动已经处于低潮。这是不能不承认的。张主席还告诫说，如果我们共产党人仍然不能从‘左’的躯壳里解放出来，这将是我们这一代最大的悲剧。” <br/>　　彭德怀有些惊讶，面前这个黄口乳子竟敢放肆地冒出这种宏论！他厌烦地把头歪在一边，下嘴唇撅着，两个嘴角弯成了一个彭德怀式的弧线，不作声了。<br/>　　黄超觉得自己有点操之过急，就站起来，对着楼梯口叫： <br/>　　“警卫员！把东西拿上来！” <br/>　　原来他带的两个警卫员等在楼下，这时闻声走了上来。一个背着一大一小两个口袭，另一个背着一个沉甸甸的皮包。黄超满脸堆笑，指着那个小口袋说：“这是几斤牛肉干，味道蛮不错的。”又指指那个大口袋说：“这是几升大米，是我们张主席从川陕带来的，这地方想找这个就太不容易了。” <br/>　　说过，他又从另一个警卫员手里接过沉甸甸的皮包，从里面取出几个包包，笑得很迷人的： <br/>　　“这是三百块白洋，只不过是张主席的一点微意。” <br/>　　彭德怀看见大米和牛肉干，还微微点了点头，一见递过来的白洋，脸色立刻变了。 <br/>　　“这是干什么！”他的语调有些严厉。 <br/>　　“也不过怕军团长手头不便……” <br/>　　彭德怀终于克制住自己，没有发作，但是他站在那里一声不响，简直象石头雕像一样冷峻。 <br/>　　黄超异常狼狈，只好慌慌张张把钱放在一个用木板搭成的桌案上。他尴尬得不知说什么好，幸亏他脑子聪敏灵活，就乓地打了个潇洒的敬礼，笑着说： <br/>　　“彭军团长，您恐怕很疲劳了，我们也该回去了。” <br/>　　彭德怀站起来，勉强点了点头。黄超带着警卫员慌乱地下楼去了。 <br/>　　直到黄超走出很远，他还觉得满心不舒服，望着这个张国焘的使者，狠狠骂道： <br/>　　“呸！什么东西！纯粹是旧军阀的一套！” <br/>　　说过，他就坐在火塘边陷入深深的沉思里。他一遍又一遍地想着这个黄口乳子的来意。 <br/>　　这时，三军团的政治委员杨尚昆走了进来，他一见彭德怀满脸怒容，就问： <br/>　　“德怀同志，黄超在这里谈什么了？” <br/>　　彭德怀的火立刻又升腾起来，他指了指桌上的白洋，骂道： <br/>　　“张国焘他把我彭德怀看成什么人了？他把我当成军阀！ <br/>　　我要当军阀，还来红军干什么？真是岂有此理！” <br/>　　“这个家伙值得警惕！”杨尚昆也沉到思索中了。 </p></div>
