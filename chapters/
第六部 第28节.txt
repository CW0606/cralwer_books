<h1 id="chapter_title_177988"><strong>第六部 第28节</strong></h1>
<p class="info">
分类：<a class="cate" href="/novel-list-23.html" target="blank">文学小说</a><span class="divide">|</span>作者：<a class="name" href="/pdnovel.php?mod=search&amp;ac=author&amp;srchtxt=周大新&amp;searchsubmit=yes"  target="_blank">周大新</a><span class="divide">|</span>发布时间：2015-04-25 22:19:49<span class="divide">|</span></p>
<div class="page-body">
<div class="page-content">
<p>火纸点燃后冒出的轻烟，陆续地在各家的祖坟头上升起来了。达志拄拐站在自家的祖坟上，默默地向田野望去。 <br/>　　四周竟有这么多的坟头，已经有这么多的人死了！我也快了吧？阎王爷给我的阳寿还有多长？我还能熬过几个清明节呢？说不定下一个清明节，我就也在这土堆里了吧？…… <br/>　　昌盛、小瑾正在他们的几位祖爷爷、父亲、妈妈和尤婶的坟头上焚燃纸钱。孙子、孙媳原是不让达志也来的，可达志担心下一个清明节自己就走不动了，执意要来坟上看看。 <br/>　　按清明节添坟的规矩，各个坟头昌盛都用铁锹新培了土，新鲜的黑土使每个坟包都有了丝生气。达志在父亲尚安业的坟前鞠了个躬，他是很想下跪的，可又担心一跪下就站不起了。爹，原谅我吧，我也老了……我晓得你还牵挂着织绸织缎的事，听说尚吉利织丝厂就要恢复生产了…… <br/>　　“爷爷，咱们回吧。”昌盛和小瑾过来搀住了他的胳膊。三个人缓缓地向地头走去。地头上停着一辆平板车，月儿陪着卓远先已经坐在板车上了——早饭后，昌盛就是用这辆板车把两位老人和小瑾、月儿拉到这田野里的。雅娴奶奶几个月前病逝，卓远今儿个执意要亲来坟上给雅娴烧纸。好在两家的坟地相离不远，昌盛可以一车拉了来。 <br/>　　“月儿，要好生照料你爷爷。”昌盛拉动板车往回返时，达志对外孙女做着交待。 <br/>　　“放心吧，达志，月儿照顾我周到着哩，有时看她那个忙劲，我就真想也随你雅娴嫂子去了作罢。”卓远叹了口气说。 <br/>　　“瞎说罢你，你不是说你还要写一本书吗？” <br/>　　“是呵，只不知道能不能写出来。” <br/>　　“书名是啥？” <br/>　　“暂时保密吧，等我写出来了，自然让你先……” <br/>　　“呜——”一辆吉普车突然在他们的板车旁停了下来，车门在众人的惊疑中打开后，承达从车上跳下。“卓大伯，爹——” <br/>　　“你这是——？” <br/>　　“我刚刚被从五七干校接回来，上级已任命我来当南阳生产指挥部的指挥长了，我现在是回城上任。” <br/>　　“生产指挥部——？” <br/>　　“就是专门指挥经济生产的部门。前两年虽然成立了，但形同虚设，这次要真正负起指挥生产的责任！” <br/>　　“这么说你们终于明白该抓什么了？！”卓远看定承达。 <br/>　　“我们浪费了太多的时间！” <br/>　　“会是真的？”达志瞪大了眼。 <br/>　　“当然！”承达点头，“尚吉利织丝厂的生产也要立刻恢复。昌盛，你是厂里的老工人了，回去后就马上到厂里，我随后也去，要先领人把厂子清理出来！” <br/>　　“不会再——？”达志抓住了承达的手腕。 <br/>　　“放心，爹，灾难已让我们每个人都聪明了许多！”承达说罢转身上了车，车子立刻挟着烟尘向城里驰去。 <br/>　　“乱极而治，衰极而兴，达志，也许我们还会看到一个新局面。” <br/>　　“会吗？” <br/>　　“一个国家的人不会长久忍耐一种吃不好、穿不好、住不好且没有安全感的生活，他们必然会努力寻找出能把他们带入富裕、安宁、幸福日子的人和制度。这是人类社会发展的规律。我们没有理由不相信局面会发生变化！”卓远在板车的缓慢移动中拍着达志的手。 <br/>　　“依你之见，尚吉利织丝厂也还有兴盛的一天？” <br/>　　“当然！” <br/>　　“我老死之前能看见这一天？” <br/>　　“怎么你也说到死？你看你还是满口好牙哩，活过一百不成问题；你爷爷当初不是活到了一百零三，你们可是长寿家族！……” <br/>　　老天保佑卓远哥说得准确，保佑我能活到尚吉利织丝厂重新兴盛的日子。爹，但愿我晚点去见你的时候，真能带给你一个好消息：尚吉利绸缎又获了“霸王”美誉…… <br/>　　星星们还赖在天上没走，最早的一声鸡啼还未翻过尚家的院墙落进院里，达志已经拄着拐杖走出了睡屋，在门口伸了一个长长的懒腰。他走到前院那块竖着的石头前，把拐杖往石头上一靠，先是张嘴呼了几口浊气，然后开始扭腰、甩手、搓头。这是他自己的健身术。几个项目做完，见星星已稀了不少，四周邻居家的鸡叫已开始热闹，就走到了昌盛、小瑾他们的睡屋门口，用拐杖把门捣捣。 <br/>　　“谁？”昌盛仿佛在忙着什么，声音有些喘。 <br/>　　“我。叫旺旺起来！” <br/>　　“爷爷，天不是还没亮？”小瑾的声音，身子似乎被什么压着。 <br/>　　“快亮了，让他起来！”听你们的声音就知道你们在干什么。傻东西们，一大早就做这种事会伤身子的！ <br/>　　“干啥子么？”是旺旺含混的嘟囔，不过，已经有穿衣系带的窸窣声了。片刻之后，旺旺拉门站在了门口：“太爷爷，有事？” <br/>　　“跟我走！”达志扭身拄杖向后院里移步，拐杖捣在地上很重，声音响亮地撞在院墙上。他最后在后院的那棵老桑树下站定，在衣兜里摸索了一阵，把一卷纸朝旺旺送去。 <br/>　　“啥？” <br/>　　“书！” <br/>　　“书？俺们学校已经发了新书。” <br/>　　“这和你们老师发的书不同，这是咱们家的书！” <br/>　　“咱们家的书，写的啥？”旺旺翻了翻，可天还黑，看不清。 <br/>　　“有关织绸织缎的事。” <br/>　　“织绸缎？我不喜欢读这书，我长大了要当科学家，造在天上飞的卫星！太爷爷，给你！” <br/>　　“拿住！” <br/>　　“拿住我也不想读！” <br/>　　“要读！” <br/>　　“我不——” <br/>　　“啪！”旺旺屁股上挨了重重一拐杖，一阵火烧火燎的疼感在向腿上蔓延。 <br/>　　“太爷爷，你凭啥打人？” <br/>　　“要读这书！” <br/>　　“我不——” <br/>　　“啪！”又一拐杖落到了旺旺的屁股上。在越来越亮的晨光里，有两颗晶亮的泪珠委委屈屈地爬出了旺旺的眼眶。 <br/>　　“要读！” <br/>　　“咋读？”旺旺哽咽着问。 <br/>　　“每天早晨读一页，还要会背！不识的字和不懂的句可以问我和你爹。头一天读的东西，第二日早上要先背一遍！” <br/>　　天已经彻底亮了。旺旺不甚情愿地翻开那本用麻线装订起来的书，将目光迟迟疑疑地放上去。 <br/>　　“最好读出声，我和你爷爷、你爹当初都是读出声的，这样也便于记住！” <br/>　　“自唐武德八年始，吾南阳尚家从丝绸织造，迄今已千三百五十一年……” <br/>　　几缕鲜红的霞光翻过院墙，静静地映在那一老一少身上……</p></div>
